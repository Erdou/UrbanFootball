# Changelog

## [1.0.2] - 2026-04-06

### Added
- **Disable Touchscreen setting** — new option to turn off tap-to-score during gameplay, preventing accidental score changes from wrist contact.
- **Multi-language support** — store listing and in-app strings now available in 36 languages (Arabic, Bulgarian, Czech, Danish, Dutch, Estonian, Finnish, French, German, Greek, Hebrew, Croatian, Hungarian, Indonesian, Italian, Japanese, Korean, Latvian, Lithuanian, Norwegian, Polish, Portuguese, Romanian, Russian, Slovak, Slovenian, Spanish, Swedish, Thai, Turkish, Ukrainian, Vietnamese, Chinese Simplified, Chinese Traditional, Malay, and English).
- **Usage guide** — detailed instructions covering app setup, scoring, goalie timer, score undo, and score adjust mode.
- **Button control cheat sheets** — quick-reference sheets for watch controls in multiple languages.

### Improved
- **Responsive layout** — game time and goalie timer positioning now adapts to different screen heights for better readability on all supported devices.
