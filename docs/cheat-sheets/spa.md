# Guia rapida de controles del reloj

## Botones fisicos

| Boton | Accion | Funcion |
|---|---|---|
| **START/STOP** (top-right) | Pulsar | Iniciar o pausar la grabacion |
| **START/STOP** (top-right) | Pulsar (en el menu de pausa) | Confirmar la opcion seleccionada (Reanudar/Guardar/Descartar) |
| **UP/MENU** (top-left) | Pulsar | +1 puntuacion izquierda |
| **UP/MENU** (top-left) | Pulsacion larga | Entrar en modo Ajuste de puntuacion (lado izquierdo seleccionado) |
| **UP/MENU** (top-left) | Pulsar (en modo Ajuste de puntuacion) | Alternar entre lado izquierdo/derecho |
| **DOWN** (bottom-left) | Pulsar | +1 puntuacion derecha |
| **DOWN** (bottom-left) | Pulsar (en modo Ajuste de puntuacion) | −1 en el lado seleccionado |
| **DOWN** (bottom-left) | Pulsar (en pantalla de confirmacion) | Volver al menu de pausa |
| **BACK/LAP** (bottom-right) | Pulsar | Reiniciar temporizador de portero |
| **BACK/LAP** (bottom-right) | Pulsacion larga | Abrir configuracion del temporizador de portero |
| **BACK/LAP** (bottom-right) | Pulsar (en modo Ajuste de puntuacion) | Salir del modo Ajuste de puntuacion |
| **BACK/LAP** (bottom-right) | Pulsar (en pantalla de confirmacion) | Volver al menu de pausa |

## Zonas tactiles

| Zona | Accion | Funcion |
|---|---|---|
| Mitad izquierda de la pantalla | Tocar | +1 puntuacion izquierda |
| Mitad derecha de la pantalla | Tocar | +1 puntuacion derecha |
| ~30% inferior de la pantalla | Tocar | Reiniciar temporizador de portero |
| Lado resaltado (durante la ventana de deshacer de 3s) | Tocar | Deshacer ultima puntuacion |
