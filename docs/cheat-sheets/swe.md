# Snabbguide för klockkontroller

## Fysiska knappar

| Knapp | Åtgärd | Funktion |
|---|---|---|
| **START/STOP** (top-right) | Tryck | Starta eller pausa inspelning |
| **START/STOP** (top-right) | Tryck (i pausmenyn) | Bekräfta valt alternativ (Återuppta/Spara/Radera) |
| **UP/MENU** (top-left) | Tryck | +1 vänster poäng |
| **UP/MENU** (top-left) | Långt tryck | Öppna poängjusteringsläge (vänster sida vald) |
| **UP/MENU** (top-left) | Tryck (i poängjustering) | Växla mellan vänster/höger sida |
| **DOWN** (bottom-left) | Tryck | +1 höger poäng |
| **DOWN** (bottom-left) | Tryck (i poängjustering) | −1 på vald sida |
| **DOWN** (bottom-left) | Tryck (i bekräftelseskärm) | Tillbaka till pausmenyn |
| **BACK/LAP** (bottom-right) | Tryck | Återställ målvaktstimer |
| **BACK/LAP** (bottom-right) | Långt tryck | Öppna målvaktstimerinställningar |
| **BACK/LAP** (bottom-right) | Tryck (i poängjustering) | Lämna poängjusteringsläge |
| **BACK/LAP** (bottom-right) | Tryck (i bekräftelseskärm) | Tillbaka till pausmenyn |

## Touchscreenzoner

| Zon | Åtgärd | Funktion |
|---|---|---|
| Vänster halva av skärmen | Tryck | +1 vänster poäng |
| Höger halva av skärmen | Tryck | +1 höger poäng |
| Nedre ~30% av skärmen | Tryck | Återställ målvaktstimer |
| Markerad sida (under 3s ångra-fönster) | Tryck | Ångra senaste poäng |
