# Saat Kontrolleri Hızlı Kılavuzu

## Fiziksel Butonlar

| Buton | Eylem | Ne yapar |
|---|---|---|
| **START/STOP** (top-right) | Bas | Kaydı başlat veya duraklat |
| **START/STOP** (top-right) | Bas (duraklatma menüsünde) | Seçili seçeneği onayla (Devam/Kaydet/Sil) |
| **UP/MENU** (top-left) | Bas | +1 sol skor |
| **UP/MENU** (top-left) | Uzun bas | Skor Düzenleme Moduna gir (sol taraf seçili) |
| **UP/MENU** (top-left) | Bas (Skor Düzenleme Modunda) | Sol/sağ taraf arasında geçiş yap |
| **DOWN** (bottom-left) | Bas | +1 sağ skor |
| **DOWN** (bottom-left) | Bas (Skor Düzenleme Modunda) | −1 seçili tarafta |
| **DOWN** (bottom-left) | Bas (onay ekranında) | Duraklatma menüsüne geri dön |
| **BACK/LAP** (bottom-right) | Bas | Kaleci zamanlayıcısını sıfırla |
| **BACK/LAP** (bottom-right) | Uzun bas | Kaleci zamanlayıcısı ayarlarını aç |
| **BACK/LAP** (bottom-right) | Bas (Skor Düzenleme Modunda) | Skor Düzenleme Modundan çık |
| **BACK/LAP** (bottom-right) | Bas (onay ekranında) | Duraklatma menüsüne geri dön |

## Dokunmatik Ekran Bölgeleri

| Bölge | Eylem | Ne yapar |
|---|---|---|
| Ekranın sol yarısı | Dokun | +1 sol skor |
| Ekranın sağ yarısı | Dokun | +1 sağ skor |
| Ekranın alt ~30% bölümü | Dokun | Kaleci zamanlayıcısını sıfırla |
| Vurgulanan taraf (3s geri alma penceresi sırasında) | Dokun | Son skoru geri al |
