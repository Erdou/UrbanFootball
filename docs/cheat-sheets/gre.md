# Σύντομος Οδηγός Χειρισμού Ρολογιού

## Φυσικά Κουμπιά

| Κουμπί | Ενέργεια | Τι κάνει |
|---|---|---|
| **START/STOP** (top-right) | Πάτημα | Έναρξη ή παύση εγγραφής |
| **START/STOP** (top-right) | Πάτημα (στο μενού παύσης) | Επιβεβαίωση επιλεγμένης ενέργειας (Συνέχεια/Αποθήκευση/Απόρριψη) |
| **UP/MENU** (top-left) | Πάτημα | +1 αριστερό σκορ |
| **UP/MENU** (top-left) | Παρατεταμένο πάτημα | Είσοδος στη Λειτουργία Προσαρμογής Σκορ (επιλεγμένη η αριστερή πλευρά) |
| **UP/MENU** (top-left) | Πάτημα (στη Λειτουργία Προσαρμογής Σκορ) | Εναλλαγή μεταξύ αριστερής/δεξιάς πλευράς |
| **DOWN** (bottom-left) | Πάτημα | +1 δεξί σκορ |
| **DOWN** (bottom-left) | Πάτημα (στη Λειτουργία Προσαρμογής Σκορ) | −1 στην επιλεγμένη πλευρά |
| **DOWN** (bottom-left) | Πάτημα (στην οθόνη επιβεβαίωσης) | Επιστροφή στο μενού παύσης |
| **BACK/LAP** (bottom-right) | Πάτημα | Επαναφορά χρονομέτρου τερματοφύλακα |
| **BACK/LAP** (bottom-right) | Παρατεταμένο πάτημα | Άνοιγμα ρυθμίσεων χρονομέτρου τερματοφύλακα |
| **BACK/LAP** (bottom-right) | Πάτημα (στη Λειτουργία Προσαρμογής Σκορ) | Έξοδος από τη Λειτουργία Προσαρμογής Σκορ |
| **BACK/LAP** (bottom-right) | Πάτημα (στην οθόνη επιβεβαίωσης) | Επιστροφή στο μενού παύσης |

## Ζώνες Οθόνης Αφής

| Ζώνη | Ενέργεια | Τι κάνει |
|---|---|---|
| Αριστερό μισό της οθόνης | Πάτημα | +1 αριστερό σκορ |
| Δεξί μισό της οθόνης | Πάτημα | +1 δεξί σκορ |
| ~30% κάτω μέρος της οθόνης | Πάτημα | Επαναφορά χρονομέτρου τερματοφύλακα |
| Επισημασμένη πλευρά (κατά τη διάρκεια παραθύρου αναίρεσης 3s) | Πάτημα | Αναίρεση τελευταίου σκορ |
