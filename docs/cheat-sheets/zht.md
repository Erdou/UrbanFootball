# 手錶操作速查表

## 實體按鈕

| 按鈕 | 操作 | 功能說明 |
|---|---|---|
| **START/STOP** (top-right) | 按下 | 開始或暫停記錄 |
| **START/STOP** (top-right) | 按下（在暫停選單中） | 確認所選項目（恢復/儲存/捨棄） |
| **UP/MENU** (top-left) | 按下 | 左側比分 +1 |
| **UP/MENU** (top-left) | 長按 | 進入比分調整模式（左側被選取） |
| **UP/MENU** (top-left) | 按下（在比分調整中） | 在左側/右側之間切換 |
| **DOWN** (bottom-left) | 按下 | 右側比分 +1 |
| **DOWN** (bottom-left) | 按下（在比分調整中） | 所選一側 −1 |
| **DOWN** (bottom-left) | 按下（在確認畫面中） | 返回暫停選單 |
| **BACK/LAP** (bottom-right) | 按下 | 重設守門員計時器 |
| **BACK/LAP** (bottom-right) | 長按 | 開啟守門員計時器設定 |
| **BACK/LAP** (bottom-right) | 按下（在比分調整中） | 退出比分調整模式 |
| **BACK/LAP** (bottom-right) | 按下（在確認畫面中） | 返回暫停選單 |

## 觸控螢幕區域

| 區域 | 操作 | 功能說明 |
|---|---|---|
| 螢幕左半部分 | 點擊 | 左側比分 +1 |
| 螢幕右半部分 | 點擊 | 右側比分 +1 |
| 螢幕底部 ~30% | 點擊 | 重設守門員計時器 |
| 醒目標示的一側（3s 復原視窗期間） | 點擊 | 復原上一次得分 |
