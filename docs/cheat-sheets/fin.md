# Kellon ohjaus - pikaopas

## Fyysiset painikkeet

| Painike | Toiminto | Mita se tekee |
|---|---|---|
| **START/STOP** (top-right) | Paina | Aloita tai keskeyta tallennus |
| **START/STOP** (top-right) | Paina (taukovalikossa) | Vahvista valittu vaihtoehto (Jatka/Tallenna/Hylkaa) |
| **UP/MENU** (top-left) | Paina | +1 vasen tulos |
| **UP/MENU** (top-left) | Pitka painallus | Siirry tuloksen muokkaustilaan (vasen puoli valittu) |
| **UP/MENU** (top-left) | Paina (tuloksen muokkaustilassa) | Vaihda vasemman/oikean puolen valilla |
| **DOWN** (bottom-left) | Paina | +1 oikea tulos |
| **DOWN** (bottom-left) | Paina (tuloksen muokkaustilassa) | −1 valitulla puolella |
| **DOWN** (bottom-left) | Paina (vahvistusnaytossa) | Takaisin taukovalikkoon |
| **BACK/LAP** (bottom-right) | Paina | Nollaa maalivahdin ajastin |
| **BACK/LAP** (bottom-right) | Pitka painallus | Avaa maalivahdin ajastimen asetukset |
| **BACK/LAP** (bottom-right) | Paina (tuloksen muokkaustilassa) | Poistu tuloksen muokkaustilasta |
| **BACK/LAP** (bottom-right) | Paina (vahvistusnaytossa) | Takaisin taukovalikkoon |

## Kosketusnayton alueet

| Alue | Toiminto | Mita se tekee |
|---|---|---|
| Nayton vasen puolisko | Napauta | +1 vasen tulos |
| Nayton oikea puolisko | Napauta | +1 oikea tulos |
| Nayton alin ~30% | Napauta | Nollaa maalivahdin ajastin |
| Korostettu puoli (3s kumoamisikkunan aikana) | Napauta | Kumoa viimeisin tulos |
