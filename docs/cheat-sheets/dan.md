# Hurtigguide til urbetjening

## Fysiske knapper

| Knap | Handling | Funktion |
|---|---|---|
| **START/STOP** (top-right) | Tryk | Start eller pause optagelse |
| **START/STOP** (top-right) | Tryk (i pausemenu) | Bekræft valgt mulighed (Genoptag/Gem/Kassér) |
| **UP/MENU** (top-left) | Tryk | +1 venstre score |
| **UP/MENU** (top-left) | Langt tryk | Åbn scorekorrektionsmodus (venstre side valgt) |
| **UP/MENU** (top-left) | Tryk (i scorekorrektion) | Skift mellem venstre/højre side |
| **DOWN** (bottom-left) | Tryk | +1 højre score |
| **DOWN** (bottom-left) | Tryk (i scorekorrektion) | −1 på valgte side |
| **DOWN** (bottom-left) | Tryk (i bekræftelsesskærm) | Tilbage til pausemenu |
| **BACK/LAP** (bottom-right) | Tryk | Nulstil målmandstimer |
| **BACK/LAP** (bottom-right) | Langt tryk | Åbn målmandstimerindstillinger |
| **BACK/LAP** (bottom-right) | Tryk (i scorekorrektion) | Forlad scorekorrektionsmodus |
| **BACK/LAP** (bottom-right) | Tryk (i bekræftelsesskærm) | Tilbage til pausemenu |

## Touchscreenzoner

| Zone | Handling | Funktion |
|---|---|---|
| Venstre halvdel af skærmen | Tryk | +1 venstre score |
| Højre halvdel af skærmen | Tryk | +1 højre score |
| Nederste ~30% af skærmen | Tryk | Nulstil målmandstimer |
| Fremhævet side (under 3s fortryd-vindue) | Tryk | Fortryd sidste score |
