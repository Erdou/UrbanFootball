# Skrocona instrukcja obslugi zegarka

## Przyciski fizyczne

| Przycisk | Akcja | Co robi |
|---|---|---|
| **START/STOP** (top-right) | Nacisnij | Rozpocznij lub wstrzymaj nagrywanie |
| **START/STOP** (top-right) | Nacisnij (w menu pauzy) | Potwierdz wybrana opcje (Wznow/Zapisz/Odrzuc) |
| **UP/MENU** (top-left) | Nacisnij | +1 wynik lewy |
| **UP/MENU** (top-left) | Dlugie nacisniecie | Wejdz w tryb korekty wyniku (wybrana lewa strona) |
| **UP/MENU** (top-left) | Nacisnij (w trybie korekty) | Przelacz miedzy lewa/prawa strona |
| **DOWN** (bottom-left) | Nacisnij | +1 wynik prawy |
| **DOWN** (bottom-left) | Nacisnij (w trybie korekty) | −1 po wybranej stronie |
| **DOWN** (bottom-left) | Nacisnij (na ekranie potwierdzenia) | Wroc do menu pauzy |
| **BACK/LAP** (bottom-right) | Nacisnij | Resetuj timer bramkarza |
| **BACK/LAP** (bottom-right) | Dlugie nacisniecie | Otworz konfiguracje timera bramkarza |
| **BACK/LAP** (bottom-right) | Nacisnij (w trybie korekty) | Wyjdz z trybu korekty wyniku |
| **BACK/LAP** (bottom-right) | Nacisnij (na ekranie potwierdzenia) | Wroc do menu pauzy |

## Strefy ekranu dotykowego

| Strefa | Akcja | Co robi |
|---|---|---|
| Lewa polowa ekranu | Dotknij | +1 wynik lewy |
| Prawa polowa ekranu | Dotknij | +1 wynik prawy |
| Dolne ~30% ekranu | Dotknij | Resetuj timer bramkarza |
| Podswietlona strona (w oknie cofania 3s) | Dotknij | Cofnij ostatni wynik |
