# Pulkstena vadiba - isa pamaciba

## Fiziskas pogas

| Poga | Darbiba | Ko ta dara |
|---|---|---|
| **START/STOP** (top-right) | Nospiest | Sakt vai apturet ierakstisanu |
| **START/STOP** (top-right) | Nospiest (pauzes izvelne) | Apstiprinat izveleeto opciju (Turpinat/Saglabaot/Atmeest) |
| **UP/MENU** (top-left) | Nospiest | +1 kreisais rezultats |
| **UP/MENU** (top-left) | Ilgi nospiest | Ieiet rezultata korigeSanas rezima (kreisa puse izveleta) |
| **UP/MENU** (top-left) | Nospiest (rezultata korigeSanas rezima) | Paarsleegt starp kreiso/labo pusi |
| **DOWN** (bottom-left) | Nospiest | +1 labais rezultats |
| **DOWN** (bottom-left) | Nospiest (rezultata korigeSanas rezima) | −1 izveleetaja puse |
| **DOWN** (bottom-left) | Nospiest (apstiprinajuma ekrana) | Atpakal uz pauzes izvelni |
| **BACK/LAP** (bottom-right) | Nospiest | Atiestatit vartsarga taimeri |
| **BACK/LAP** (bottom-right) | Ilgi nospiest | Atvert vartsarga taimera iestatijumus |
| **BACK/LAP** (bottom-right) | Nospiest (rezultata korigeSanas rezima) | Iziet no rezultata korigeSanas rezima |
| **BACK/LAP** (bottom-right) | Nospiest (apstiprinajuma ekrana) | Atpakal uz pauzes izvelni |

## Skarienekrana zonas

| Zona | Darbiba | Ko ta dara |
|---|---|---|
| Ekrana kreisa puse | Pieskarties | +1 kreisais rezultats |
| Ekrana laba puse | Pieskarties | +1 labais rezultats |
| Ekrana apakSejie ~30% | Pieskarties | Atiestatit vartsarga taimeri |
| IzcelTa puse (3s atsiSanas loga laika) | Pieskarties | Atsaukt pedejo rezultatu |
