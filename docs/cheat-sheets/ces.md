# Tahak k ovladani hodinek

## Fyzicka tlacitka

| Tlacitko | Akce | Co to dela |
|---|---|---|
| **START/STOP** (top-right) | Stisknete | Spustit nebo pozastavit zaznam |
| **START/STOP** (top-right) | Stisknete (v menu pauzy) | Potvrdit vybranou moznost (Pokracovat/Ulozit/Zahodit) |
| **UP/MENU** (top-left) | Stisknete | +1 leve skore |
| **UP/MENU** (top-left) | Dlouhy stisk | Vstoupit do rezimu upravy skore (vybrana leva strana) |
| **UP/MENU** (top-left) | Stisknete (v rezimu upravy) | Prepnout mezi levou/pravou stranou |
| **DOWN** (bottom-left) | Stisknete | +1 prave skore |
| **DOWN** (bottom-left) | Stisknete (v rezimu upravy) | −1 na vybrane strane |
| **DOWN** (bottom-left) | Stisknete (na potvrzovaci obrazovce) | Zpet do menu pauzy |
| **BACK/LAP** (bottom-right) | Stisknete | Resetovat casovac brankaре |
| **BACK/LAP** (bottom-right) | Dlouhy stisk | Otevrit konfiguraci casovace brankare |
| **BACK/LAP** (bottom-right) | Stisknete (v rezimu upravy) | Opustit rezim upravy skore |
| **BACK/LAP** (bottom-right) | Stisknete (na potvrzovaci obrazovce) | Zpet do menu pauzy |

## Zony dotykove obrazovky

| Zona | Akce | Co to dela |
|---|---|---|
| Leva polovina obrazovky | Klepnete | +1 leve skore |
| Prava polovina obrazovky | Klepnete | +1 prave skore |
| Dolnich ~30% obrazovky | Klepnete | Resetovat casovac brankare |
| Zvyraznena strana (behem 3s okna pro vraceni) | Klepnete | Vratit posledni skore |
