# Lembar Panduan Kontrol Jam Tangan

## Tombol Fisik

| Tombol | Tindakan | Fungsi |
|---|---|---|
| **START/STOP** (top-right) | Tekan | Mulai atau jeda perekaman |
| **START/STOP** (top-right) | Tekan (di menu jeda) | Konfirmasi pilihan (Lanjutkan/Simpan/Buang) |
| **UP/MENU** (top-left) | Tekan | +1 skor kiri |
| **UP/MENU** (top-left) | Tekan lama | Masuk Mode Sesuaikan Skor (sisi kiri terpilih) |
| **UP/MENU** (top-left) | Tekan (di Mode Sesuaikan Skor) | Beralih antara sisi kiri/kanan |
| **DOWN** (bottom-left) | Tekan | +1 skor kanan |
| **DOWN** (bottom-left) | Tekan (di Mode Sesuaikan Skor) | −1 pada sisi yang dipilih |
| **DOWN** (bottom-left) | Tekan (di layar konfirmasi) | Kembali ke menu jeda |
| **BACK/LAP** (bottom-right) | Tekan | Reset timer penjaga gawang |
| **BACK/LAP** (bottom-right) | Tekan lama | Buka pengaturan timer penjaga gawang |
| **BACK/LAP** (bottom-right) | Tekan (di Mode Sesuaikan Skor) | Keluar Mode Sesuaikan Skor |
| **BACK/LAP** (bottom-right) | Tekan (di layar konfirmasi) | Kembali ke menu jeda |

## Zona Layar Sentuh

| Zona | Tindakan | Fungsi |
|---|---|---|
| Separuh kiri layar | Ketuk | +1 skor kiri |
| Separuh kanan layar | Ketuk | +1 skor kanan |
| ~30% bagian bawah layar | Ketuk | Reset timer penjaga gawang |
| Sisi yang disorot (selama jendela pembatalan 3s) | Ketuk | Batalkan skor terakhir |
