# Hurtigguide for klokkeknapper

## Fysiske knapper

| Knapp | Handling | Funksjon |
|---|---|---|
| **START/STOP** (top-right) | Trykk | Start eller pause opptak |
| **START/STOP** (top-right) | Trykk (i pausemeny) | Bekreft valgt alternativ (Gjenoppta/Lagre/Forkast) |
| **UP/MENU** (top-left) | Trykk | +1 venstre poengsum |
| **UP/MENU** (top-left) | Langt trykk | Åpne poengkorrigeringsmodus (venstre side valgt) |
| **UP/MENU** (top-left) | Trykk (i poengkorrigering) | Veksle mellom venstre/høyre side |
| **DOWN** (bottom-left) | Trykk | +1 høyre poengsum |
| **DOWN** (bottom-left) | Trykk (i poengkorrigering) | −1 på valgt side |
| **DOWN** (bottom-left) | Trykk (i bekreftelsesskjerm) | Tilbake til pausemeny |
| **BACK/LAP** (bottom-right) | Trykk | Nullstill keepertimer |
| **BACK/LAP** (bottom-right) | Langt trykk | Åpne keepertimerinnstillinger |
| **BACK/LAP** (bottom-right) | Trykk (i poengkorrigering) | Forlat poengkorrigeringsmodus |
| **BACK/LAP** (bottom-right) | Trykk (i bekreftelsesskjerm) | Tilbake til pausemeny |

## Berøringsskjermområder

| Område | Handling | Funksjon |
|---|---|---|
| Venstre halvdel av skjermen | Trykk | +1 venstre poengsum |
| Høyre halvdel av skjermen | Trykk | +1 høyre poengsum |
| Nederste ~30% av skjermen | Trykk | Nullstill keepertimer |
| Uthevet side (under 3s angre-vindu) | Trykk | Angre siste poengsum |
