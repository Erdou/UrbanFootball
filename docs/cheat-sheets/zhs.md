# 手表操作速查表

## 物理按钮

| 按钮 | 操作 | 功能说明 |
|---|---|---|
| **START/STOP** (top-right) | 按下 | 开始或暂停记录 |
| **START/STOP** (top-right) | 按下（在暂停菜单中） | 确认所选项目（恢复/保存/放弃） |
| **UP/MENU** (top-left) | 按下 | 左侧比分 +1 |
| **UP/MENU** (top-left) | 长按 | 进入比分调整模式（左侧被选中） |
| **UP/MENU** (top-left) | 按下（在比分调整中） | 在左侧/右侧之间切换 |
| **DOWN** (bottom-left) | 按下 | 右侧比分 +1 |
| **DOWN** (bottom-left) | 按下（在比分调整中） | 所选一侧 −1 |
| **DOWN** (bottom-left) | 按下（在确认界面中） | 返回暂停菜单 |
| **BACK/LAP** (bottom-right) | 按下 | 重置守门员计时器 |
| **BACK/LAP** (bottom-right) | 长按 | 打开守门员计时器设置 |
| **BACK/LAP** (bottom-right) | 按下（在比分调整中） | 退出比分调整模式 |
| **BACK/LAP** (bottom-right) | 按下（在确认界面中） | 返回暂停菜单 |

## 触摸屏区域

| 区域 | 操作 | 功能说明 |
|---|---|---|
| 屏幕左半部分 | 点击 | 左侧比分 +1 |
| 屏幕右半部分 | 点击 | 右侧比分 +1 |
| 屏幕底部 ~30% | 点击 | 重置守门员计时器 |
| 高亮显示的一侧（3s 撤销窗口期间） | 点击 | 撤销上一次得分 |
