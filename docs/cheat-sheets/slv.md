# Hitra navodila za upravljanje ure

## Fizicni gumbi

| Gumb | Dejanje | Kaj naredi |
|---|---|---|
| **START/STOP** (top-right) | Pritisnite | Zacni ali zaustavi snemanje |
| **START/STOP** (top-right) | Pritisnite (v meniju premora) | Potrdi izbrano moznost (Nadaljuj/Shrani/Zavrzi) |
| **UP/MENU** (top-left) | Pritisnite | +1 levi rezultat |
| **UP/MENU** (top-left) | Dolg pritisk | Vstop v nacin prilagajanja rezultata (izbrana leva stran) |
| **UP/MENU** (top-left) | Pritisnite (v nacinu prilagajanja) | Preklop med levo/desno stranjo |
| **DOWN** (bottom-left) | Pritisnite | +1 desni rezultat |
| **DOWN** (bottom-left) | Pritisnite (v nacinu prilagajanja) | −1 na izbrani strani |
| **DOWN** (bottom-left) | Pritisnite (na potrditvenem zaslonu) | Nazaj v meni premora |
| **BACK/LAP** (bottom-right) | Pritisnite | Ponastavi casovnik vratarja |
| **BACK/LAP** (bottom-right) | Dolg pritisk | Odpri nastavitve casovnika vratarja |
| **BACK/LAP** (bottom-right) | Pritisnite (v nacinu prilagajanja) | Izhod iz nacina prilagajanja rezultata |
| **BACK/LAP** (bottom-right) | Pritisnite (na potrditvenem zaslonu) | Nazaj v meni premora |

## Obmocja zaslona na dotik

| Obmocje | Dejanje | Kaj naredi |
|---|---|---|
| Leva polovica zaslona | Tapnite | +1 levi rezultat |
| Desna polovica zaslona | Tapnite | +1 desni rezultat |
| Spodnjih ~30% zaslona | Tapnite | Ponastavi casovnik vratarja |
| Oznacena stran (med 3s oknom za razveljavitev) | Tapnite | Razveljavi zadnji rezultat |
