# Guida rapida dei comandi dell'orologio

## Pulsanti fisici

| Pulsante | Azione | Funzione |
|---|---|---|
| **START/STOP** (top-right) | Premere | Avviare o mettere in pausa la registrazione |
| **START/STOP** (top-right) | Premere (nel menu pausa) | Confermare l'opzione selezionata (Riprendi/Salva/Elimina) |
| **UP/MENU** (top-left) | Premere | +1 punteggio sinistro |
| **UP/MENU** (top-left) | Pressione prolungata | Entrare in modalita Regolazione punteggio (lato sinistro selezionato) |
| **UP/MENU** (top-left) | Premere (in modalita Regolazione punteggio) | Alternare tra lato sinistro/destro |
| **DOWN** (bottom-left) | Premere | +1 punteggio destro |
| **DOWN** (bottom-left) | Premere (in modalita Regolazione punteggio) | −1 sul lato selezionato |
| **DOWN** (bottom-left) | Premere (nella schermata di conferma) | Tornare al menu pausa |
| **BACK/LAP** (bottom-right) | Premere | Reimpostare il timer del portiere |
| **BACK/LAP** (bottom-right) | Pressione prolungata | Aprire la configurazione del timer del portiere |
| **BACK/LAP** (bottom-right) | Premere (in modalita Regolazione punteggio) | Uscire dalla modalita Regolazione punteggio |
| **BACK/LAP** (bottom-right) | Premere (nella schermata di conferma) | Tornare al menu pausa |

## Zone touch

| Zona | Azione | Funzione |
|---|---|---|
| Meta sinistra dello schermo | Toccare | +1 punteggio sinistro |
| Meta destra dello schermo | Toccare | +1 punteggio destro |
| ~30% inferiore dello schermo | Toccare | Reimpostare il timer del portiere |
| Lato evidenziato (durante la finestra di annullamento di 3s) | Toccare | Annullare l'ultimo punteggio |
