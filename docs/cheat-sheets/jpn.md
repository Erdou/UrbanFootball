# ウォッチ操作チートシート

## 物理ボタン

| ボタン | 操作 | 動作内容 |
|---|---|---|
| **START/STOP** (top-right) | 押す | 記録の開始または一時停止 |
| **START/STOP** (top-right) | 押す（一時停止メニュー内） | 選択した項目を確定（再開/保存/破棄） |
| **UP/MENU** (top-left) | 押す | 左スコア +1 |
| **UP/MENU** (top-left) | 長押し | スコア調整モードに入る（左側が選択状態） |
| **UP/MENU** (top-left) | 押す（スコア調整中） | 左側/右側を切り替え |
| **DOWN** (bottom-left) | 押す | 右スコア +1 |
| **DOWN** (bottom-left) | 押す（スコア調整中） | 選択中の側を −1 |
| **DOWN** (bottom-left) | 押す（確認画面内） | 一時停止メニューに戻る |
| **BACK/LAP** (bottom-right) | 押す | ゴールキーパータイマーをリセット |
| **BACK/LAP** (bottom-right) | 長押し | ゴールキーパータイマー設定を開く |
| **BACK/LAP** (bottom-right) | 押す（スコア調整中） | スコア調整モードを終了 |
| **BACK/LAP** (bottom-right) | 押す（確認画面内） | 一時停止メニューに戻る |

## タッチスクリーンゾーン

| ゾーン | 操作 | 動作内容 |
|---|---|---|
| 画面の左半分 | タップ | 左スコア +1 |
| 画面の右半分 | タップ | 右スコア +1 |
| 画面の下部 ~30% | タップ | ゴールキーパータイマーをリセット |
| ハイライトされた側（3s 取り消しウィンドウ中） | タップ | 最後のスコアを取り消し |
