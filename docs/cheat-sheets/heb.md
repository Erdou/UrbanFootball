# דף עזר לפקדי השעון

## כפתורים פיזיים

| כפתור | פעולה | מה זה עושה |
|---|---|---|
| **START/STOP** (top-right) | לחץ | התחלה או השהיה של ההקלטה |
| **START/STOP** (top-right) | לחץ (בתפריט ההשהיה) | אישור האפשרות שנבחרה (המשך/שמור/מחק) |
| **UP/MENU** (top-left) | לחץ | +1 תוצאה שמאלית |
| **UP/MENU** (top-left) | לחיצה ארוכה | כניסה למצב עריכת תוצאה (הצד השמאלי נבחר) |
| **UP/MENU** (top-left) | לחץ (במצב עריכת תוצאה) | מעבר בין הצד השמאלי לימני |
| **DOWN** (bottom-left) | לחץ | +1 תוצאה ימנית |
| **DOWN** (bottom-left) | לחץ (במצב עריכת תוצאה) | −1 בצד הנבחר |
| **DOWN** (bottom-left) | לחץ (במסך האישור) | חזרה לתפריט ההשהיה |
| **BACK/LAP** (bottom-right) | לחץ | איפוס טיימר השוער |
| **BACK/LAP** (bottom-right) | לחיצה ארוכה | פתיחת הגדרות טיימר השוער |
| **BACK/LAP** (bottom-right) | לחץ (במצב עריכת תוצאה) | יציאה ממצב עריכת תוצאה |
| **BACK/LAP** (bottom-right) | לחץ (במסך האישור) | חזרה לתפריט ההשהיה |

## אזורי מסך מגע

| אזור | פעולה | מה זה עושה |
|---|---|---|
| חצי שמאלי של המסך | הקש | +1 תוצאה שמאלית |
| חצי ימני של המסך | הקש | +1 תוצאה ימנית |
| ~30% התחתונים של המסך | הקש | איפוס טיימר השוער |
| הצד המודגש (בחלון ביטול של 3s) | הקש | ביטול התוצאה האחרונה |
