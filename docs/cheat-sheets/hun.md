# Oravezerlok - puskak

## Fizikai gombok

| Gomb | Muvelet | Mit csinal |
|---|---|---|
| **START/STOP** (top-right) | Nyomd meg | Rogzites inditasa vagy szu"neteltetese |
| **START/STOP** (top-right) | Nyomd meg (szu"net menu"ben) | Kivalasztott opcio megerositese (Folytatas/Mentes/Elvetes) |
| **UP/MENU** (top-left) | Nyomd meg | +1 bal oldali eredmeny |
| **UP/MENU** (top-left) | Hosszu nyomas | Belepees az eredmeny modositasi modba (bal oldal kivalasztva) |
| **UP/MENU** (top-left) | Nyomd meg (eredmeny modositasi modban) | Valtas a bal/jobb oldal kozo"tt |
| **DOWN** (bottom-left) | Nyomd meg | +1 jobb oldali eredmeny |
| **DOWN** (bottom-left) | Nyomd meg (eredmeny modositasi modban) | −1 a kivalasztott oldalon |
| **DOWN** (bottom-left) | Nyomd meg (megerosito kepernyoen) | Vissza a szu"net menu"be |
| **BACK/LAP** (bottom-right) | Nyomd meg | Kapus idozito visszaallitasa |
| **BACK/LAP** (bottom-right) | Hosszu nyomas | Kapus idozito beallitasainak megnyitasa |
| **BACK/LAP** (bottom-right) | Nyomd meg (eredmeny modositasi modban) | Kilepes az eredmeny modositasi modbol |
| **BACK/LAP** (bottom-right) | Nyomd meg (megerosito kepernyoen) | Vissza a szu"net menu"be |

## Erintokepernyo zonai

| Zona | Muvelet | Mit csinal |
|---|---|---|
| Kepernyo bal fele | Erintsd meg | +1 bal oldali eredmeny |
| Kepernyo jobb fele | Erintsd meg | +1 jobb oldali eredmeny |
| Kepernyo also ~30%-a | Erintsd meg | Kapus idozito visszaallitasa |
| Kiemelt oldal (3s visszavonas ablakban) | Erintsd meg | Utolso eredmeny visszavonasa |
