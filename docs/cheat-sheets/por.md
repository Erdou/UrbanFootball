# Guia rapido de controles do relogio

## Botoes fisicos

| Botao | Acao | Funcao |
|---|---|---|
| **START/STOP** (top-right) | Pressionar | Iniciar ou pausar a gravacao |
| **START/STOP** (top-right) | Pressionar (no menu de pausa) | Confirmar a opcao selecionada (Retomar/Salvar/Descartar) |
| **UP/MENU** (top-left) | Pressionar | +1 pontuacao esquerda |
| **UP/MENU** (top-left) | Pressao longa | Entrar no modo Ajuste de pontuacao (lado esquerdo selecionado) |
| **UP/MENU** (top-left) | Pressionar (no modo Ajuste de pontuacao) | Alternar entre lado esquerdo/direito |
| **DOWN** (bottom-left) | Pressionar | +1 pontuacao direita |
| **DOWN** (bottom-left) | Pressionar (no modo Ajuste de pontuacao) | −1 no lado selecionado |
| **DOWN** (bottom-left) | Pressionar (na tela de confirmacao) | Voltar ao menu de pausa |
| **BACK/LAP** (bottom-right) | Pressionar | Reiniciar o temporizador do goleiro |
| **BACK/LAP** (bottom-right) | Pressao longa | Abrir a configuracao do temporizador do goleiro |
| **BACK/LAP** (bottom-right) | Pressionar (no modo Ajuste de pontuacao) | Sair do modo Ajuste de pontuacao |
| **BACK/LAP** (bottom-right) | Pressionar (na tela de confirmacao) | Voltar ao menu de pausa |

## Zonas de toque

| Zona | Acao | Funcao |
|---|---|---|
| Metade esquerda da tela | Tocar | +1 pontuacao esquerda |
| Metade direita da tela | Tocar | +1 pontuacao direita |
| ~30% inferior da tela | Tocar | Reiniciar o temporizador do goleiro |
| Lado destacado (durante a janela de desfazer de 3s) | Tocar | Desfazer ultima pontuacao |
