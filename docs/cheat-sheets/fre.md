# Aide-memoire des commandes de la montre

## Boutons physiques

| Bouton | Action | Fonction |
|---|---|---|
| **START/STOP** (top-right) | Appuyer | Demarrer ou mettre en pause l'enregistrement |
| **START/STOP** (top-right) | Appuyer (dans le menu pause) | Confirmer l'option selectionnee (Reprendre/Sauvegarder/Supprimer) |
| **UP/MENU** (top-left) | Appuyer | +1 score gauche |
| **UP/MENU** (top-left) | Appui long | Entrer en mode Ajustement du score (cote gauche selectionne) |
| **UP/MENU** (top-left) | Appuyer (en mode Ajustement du score) | Basculer entre le cote gauche/droit |
| **DOWN** (bottom-left) | Appuyer | +1 score droit |
| **DOWN** (bottom-left) | Appuyer (en mode Ajustement du score) | −1 sur le cote selectionne |
| **DOWN** (bottom-left) | Appuyer (sur l'ecran de confirmation) | Revenir au menu pause |
| **BACK/LAP** (bottom-right) | Appuyer | Reinitialiser le chrono gardien |
| **BACK/LAP** (bottom-right) | Appui long | Ouvrir la configuration du chrono gardien |
| **BACK/LAP** (bottom-right) | Appuyer (en mode Ajustement du score) | Quitter le mode Ajustement du score |
| **BACK/LAP** (bottom-right) | Appuyer (sur l'ecran de confirmation) | Revenir au menu pause |

## Zones tactiles

| Zone | Action | Fonction |
|---|---|---|
| Moitie gauche de l'ecran | Appuyer | +1 score gauche |
| Moitie droite de l'ecran | Appuyer | +1 score droit |
| ~30% inferieur de l'ecran | Appuyer | Reinitialiser le chrono gardien |
| Cote en surbrillance (pendant la fenetre d'annulation de 3s) | Appuyer | Annuler le dernier score |
