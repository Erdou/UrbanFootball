# Laikrodzio valdymas - trumpa atmintine

## Fiziniai mygtukai

| Mygtukas | Veiksmas | Ka tai veikia |
|---|---|---|
| **START/STOP** (top-right) | Paspauskite | Pradeti arba pristabdyti irasyma |
| **START/STOP** (top-right) | Paspauskite (pauzes meniu) | Patvirtinti pasirinkta parinkti (Testi/Issaugoti/Atmesti) |
| **UP/MENU** (top-left) | Paspauskite | +1 kairysis rezultatas |
| **UP/MENU** (top-left) | Ilgas paspaudimas | Ijungti rezultato koregavimo rezima (pasirinkta kaire puse) |
| **UP/MENU** (top-left) | Paspauskite (rezultato koregavimo rezime) | Perjungti tarp kaires/desines puses |
| **DOWN** (bottom-left) | Paspauskite | +1 desinysis rezultatas |
| **DOWN** (bottom-left) | Paspauskite (rezultato koregavimo rezime) | −1 pasirinktoje puseje |
| **DOWN** (bottom-left) | Paspauskite (patvirtinimo ekrane) | Grizti i pauzes meniu |
| **BACK/LAP** (bottom-right) | Paspauskite | Atstatyti vartininko laikmati |
| **BACK/LAP** (bottom-right) | Ilgas paspaudimas | Atidaryti vartininko laikmacio nustatymus |
| **BACK/LAP** (bottom-right) | Paspauskite (rezultato koregavimo rezime) | Iseiti is rezultato koregavimo rezimo |
| **BACK/LAP** (bottom-right) | Paspauskite (patvirtinimo ekrane) | Grizti i pauzes meniu |

## Jutiklinio ekrano zonos

| Zona | Veiksmas | Ka tai veikia |
|---|---|---|
| Kaire ekrano puse | Bakstelekite | +1 kairysis rezultatas |
| Desine ekrano puse | Bakstelekite | +1 desinysis rezultatas |
| Apatiniai ~30% ekrano | Bakstelekite | Atstatyti vartininko laikmati |
| Paryskinta puse (3s atsiSaukimo lango metu) | Bakstelekite | Atsaukti paskutini rezultata |
