# ورقة مرجعية لأزرار الساعة

## الأزرار الفعلية

| الزر | الإجراء | الوظيفة |
|---|---|---|
| **START/STOP** (top-right) | اضغط | بدء أو إيقاف التسجيل مؤقتًا |
| **START/STOP** (top-right) | اضغط (في قائمة الإيقاف المؤقت) | تأكيد الخيار المحدد (استئناف/حفظ/تجاهل) |
| **UP/MENU** (top-left) | اضغط | +1 النتيجة اليسرى |
| **UP/MENU** (top-left) | اضغط مطولًا | الدخول إلى وضع تعديل النتيجة (الجانب الأيسر محدد) |
| **UP/MENU** (top-left) | اضغط (في وضع تعديل النتيجة) | التبديل بين الجانب الأيسر والأيمن |
| **DOWN** (bottom-left) | اضغط | +1 النتيجة اليمنى |
| **DOWN** (bottom-left) | اضغط (في وضع تعديل النتيجة) | −1 على الجانب المحدد |
| **DOWN** (bottom-left) | اضغط (في شاشة التأكيد) | العودة إلى قائمة الإيقاف المؤقت |
| **BACK/LAP** (bottom-right) | اضغط | إعادة تعيين مؤقت حارس المرمى |
| **BACK/LAP** (bottom-right) | اضغط مطولًا | فتح إعدادات مؤقت حارس المرمى |
| **BACK/LAP** (bottom-right) | اضغط (في وضع تعديل النتيجة) | الخروج من وضع تعديل النتيجة |
| **BACK/LAP** (bottom-right) | اضغط (في شاشة التأكيد) | العودة إلى قائمة الإيقاف المؤقت |

## مناطق الشاشة اللمسية

| المنطقة | الإجراء | الوظيفة |
|---|---|---|
| النصف الأيسر من الشاشة | انقر | +1 النتيجة اليسرى |
| النصف الأيمن من الشاشة | انقر | +1 النتيجة اليمنى |
| ~30% السفلية من الشاشة | انقر | إعادة تعيين مؤقت حارس المرمى |
| الجانب المُظلّل (خلال نافذة التراجع 3s) | انقر | التراجع عن آخر نتيجة |
