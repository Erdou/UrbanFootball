# 워치 조작 치트시트

## 물리 버튼

| 버튼 | 동작 | 설명 |
|---|---|---|
| **START/STOP** (top-right) | 누르기 | 기록 시작 또는 일시정지 |
| **START/STOP** (top-right) | 누르기 (일시정지 메뉴에서) | 선택한 항목 확인 (재개/저장/삭제) |
| **UP/MENU** (top-left) | 누르기 | 왼쪽 점수 +1 |
| **UP/MENU** (top-left) | 길게 누르기 | 점수 조정 모드 진입 (왼쪽 선택됨) |
| **UP/MENU** (top-left) | 누르기 (점수 조정 중) | 왼쪽/오른쪽 전환 |
| **DOWN** (bottom-left) | 누르기 | 오른쪽 점수 +1 |
| **DOWN** (bottom-left) | 누르기 (점수 조정 중) | 선택된 쪽 −1 |
| **DOWN** (bottom-left) | 누르기 (확인 화면에서) | 일시정지 메뉴로 돌아가기 |
| **BACK/LAP** (bottom-right) | 누르기 | 골키퍼 타이머 초기화 |
| **BACK/LAP** (bottom-right) | 길게 누르기 | 골키퍼 타이머 설정 열기 |
| **BACK/LAP** (bottom-right) | 누르기 (점수 조정 중) | 점수 조정 모드 종료 |
| **BACK/LAP** (bottom-right) | 누르기 (확인 화면에서) | 일시정지 메뉴로 돌아가기 |

## 터치스크린 영역

| 영역 | 동작 | 설명 |
|---|---|---|
| 화면 왼쪽 절반 | 탭 | 왼쪽 점수 +1 |
| 화면 오른쪽 절반 | 탭 | 오른쪽 점수 +1 |
| 화면 하단 ~30% | 탭 | 골키퍼 타이머 초기화 |
| 강조 표시된 쪽 (3s 실행취소 창 동안) | 탭 | 마지막 점수 실행취소 |
