# Helaian Panduan Kawalan Jam Tangan

## Butang Fizikal

| Butang | Tindakan | Fungsi |
|---|---|---|
| **START/STOP** (top-right) | Tekan | Mula atau jeda rakaman |
| **START/STOP** (top-right) | Tekan (dalam menu jeda) | Sahkan pilihan (Sambung/Simpan/Buang) |
| **UP/MENU** (top-left) | Tekan | +1 skor kiri |
| **UP/MENU** (top-left) | Tekan lama | Masuk Mod Laras Skor (sebelah kiri dipilih) |
| **UP/MENU** (top-left) | Tekan (dalam Mod Laras Skor) | Tukar antara sebelah kiri/kanan |
| **DOWN** (bottom-left) | Tekan | +1 skor kanan |
| **DOWN** (bottom-left) | Tekan (dalam Mod Laras Skor) | −1 pada sebelah yang dipilih |
| **DOWN** (bottom-left) | Tekan (dalam skrin pengesahan) | Kembali ke menu jeda |
| **BACK/LAP** (bottom-right) | Tekan | Set semula pemasa penjaga gol |
| **BACK/LAP** (bottom-right) | Tekan lama | Buka tetapan pemasa penjaga gol |
| **BACK/LAP** (bottom-right) | Tekan (dalam Mod Laras Skor) | Keluar Mod Laras Skor |
| **BACK/LAP** (bottom-right) | Tekan (dalam skrin pengesahan) | Kembali ke menu jeda |

## Zon Skrin Sentuh

| Zon | Tindakan | Fungsi |
|---|---|---|
| Separuh kiri skrin | Ketik | +1 skor kiri |
| Separuh kanan skrin | Ketik | +1 skor kanan |
| ~30% bahagian bawah skrin | Ketik | Set semula pemasa penjaga gol |
| Sebelah yang diserlahkan (semasa tetingkap buat asal 3s) | Ketik | Buat asal skor terakhir |
