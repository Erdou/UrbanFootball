# Bảng tóm tắt thao tác đồng hồ

## Nút vật lý

| Nút | Thao tác | Chức năng |
|---|---|---|
| **START/STOP** (top-right) | Nhấn | Bắt đầu hoặc tạm dừng ghi |
| **START/STOP** (top-right) | Nhấn (trong menu tạm dừng) | Xác nhận lựa chọn (Tiếp tục/Lưu/Hủy) |
| **UP/MENU** (top-left) | Nhấn | +1 điểm bên trái |
| **UP/MENU** (top-left) | Nhấn giữ | Vào chế độ điều chỉnh điểm (chọn bên trái) |
| **UP/MENU** (top-left) | Nhấn (trong chế độ điều chỉnh điểm) | Chuyển đổi giữa bên trái/phải |
| **DOWN** (bottom-left) | Nhấn | +1 điểm bên phải |
| **DOWN** (bottom-left) | Nhấn (trong chế độ điều chỉnh điểm) | −1 bên đang chọn |
| **DOWN** (bottom-left) | Nhấn (trong màn hình xác nhận) | Quay lại menu tạm dừng |
| **BACK/LAP** (bottom-right) | Nhấn | Đặt lại bộ hẹn giờ thủ môn |
| **BACK/LAP** (bottom-right) | Nhấn giữ | Mở cài đặt bộ hẹn giờ thủ môn |
| **BACK/LAP** (bottom-right) | Nhấn (trong chế độ điều chỉnh điểm) | Thoát chế độ điều chỉnh điểm |
| **BACK/LAP** (bottom-right) | Nhấn (trong màn hình xác nhận) | Quay lại menu tạm dừng |

## Vùng màn hình cảm ứng

| Vùng | Thao tác | Chức năng |
|---|---|---|
| Nửa trái màn hình | Chạm | +1 điểm bên trái |
| Nửa phải màn hình | Chạm | +1 điểm bên phải |
| ~30% phía dưới màn hình | Chạm | Đặt lại bộ hẹn giờ thủ môn |
| Bên được tô sáng (trong cửa sổ hoàn tác 3s) | Chạm | Hoàn tác điểm vừa ghi |
