# คู่มือการใช้งานนาฬิกาฉบับย่อ

## ปุ่มกด

| ปุ่ม | การกระทำ | สิ่งที่เกิดขึ้น |
|---|---|---|
| **START/STOP** (top-right) | กด | เริ่มหรือหยุดพักการบันทึก |
| **START/STOP** (top-right) | กด (ในเมนูหยุดพัก) | ยืนยันตัวเลือกที่เลือก (ทำต่อ/บันทึก/ทิ้ง) |
| **UP/MENU** (top-left) | กด | คะแนนฝั่งซ้าย +1 |
| **UP/MENU** (top-left) | กดค้าง | เข้าสู่โหมดปรับคะแนน (เลือกฝั่งซ้าย) |
| **UP/MENU** (top-left) | กด (ในโหมดปรับคะแนน) | สลับระหว่างฝั่งซ้าย/ขวา |
| **DOWN** (bottom-left) | กด | คะแนนฝั่งขวา +1 |
| **DOWN** (bottom-left) | กด (ในโหมดปรับคะแนน) | −1 ฝั่งที่เลือก |
| **DOWN** (bottom-left) | กด (ในหน้าจอยืนยัน) | กลับไปที่เมนูหยุดพัก |
| **BACK/LAP** (bottom-right) | กด | รีเซ็ตตัวจับเวลาผู้รักษาประตู |
| **BACK/LAP** (bottom-right) | กดค้าง | เปิดการตั้งค่าตัวจับเวลาผู้รักษาประตู |
| **BACK/LAP** (bottom-right) | กด (ในโหมดปรับคะแนน) | ออกจากโหมดปรับคะแนน |
| **BACK/LAP** (bottom-right) | กด (ในหน้าจอยืนยัน) | กลับไปที่เมนูหยุดพัก |

## โซนหน้าจอสัมผัส

| โซน | การกระทำ | สิ่งที่เกิดขึ้น |
|---|---|---|
| ครึ่งซ้ายของหน้าจอ | แตะ | คะแนนฝั่งซ้าย +1 |
| ครึ่งขวาของหน้าจอ | แตะ | คะแนนฝั่งขวา +1 |
| ด้านล่าง ~30% ของหน้าจอ | แตะ | รีเซ็ตตัวจับเวลาผู้รักษาประตู |
| ฝั่งที่ไฮไลต์ (ระหว่างหน้าต่างยกเลิก 3s) | แตะ | ยกเลิกคะแนนล่าสุด |
