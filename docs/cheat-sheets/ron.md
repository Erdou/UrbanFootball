# Ghid rapid de comenzi pentru ceas

## Butoane fizice

| Buton | Actiune | Functie |
|---|---|---|
| **START/STOP** (top-right) | Apasare | Pornire sau pauza inregistrarii |
| **START/STOP** (top-right) | Apasare (in meniul de pauza) | Confirma optiunea selectata (Reluare/Salvare/Renuntare) |
| **UP/MENU** (top-left) | Apasare | +1 scor stanga |
| **UP/MENU** (top-left) | Apasare lunga | Intrare in modul Ajustare scor (partea stanga selectata) |
| **UP/MENU** (top-left) | Apasare (in modul Ajustare scor) | Comutare intre partea stanga/dreapta |
| **DOWN** (bottom-left) | Apasare | +1 scor dreapta |
| **DOWN** (bottom-left) | Apasare (in modul Ajustare scor) | −1 pe partea selectata |
| **DOWN** (bottom-left) | Apasare (pe ecranul de confirmare) | Inapoi la meniul de pauza |
| **BACK/LAP** (bottom-right) | Apasare | Resetare cronometru portar |
| **BACK/LAP** (bottom-right) | Apasare lunga | Deschidere configurare cronometru portar |
| **BACK/LAP** (bottom-right) | Apasare (in modul Ajustare scor) | Iesire din modul Ajustare scor |
| **BACK/LAP** (bottom-right) | Apasare (pe ecranul de confirmare) | Inapoi la meniul de pauza |

## Zone tactile

| Zona | Actiune | Functie |
|---|---|---|
| Jumatatea stanga a ecranului | Atingere | +1 scor stanga |
| Jumatatea dreapta a ecranului | Atingere | +1 scor dreapta |
| ~30% inferior al ecranului | Atingere | Resetare cronometru portar |
| Partea evidentiata (in fereastra de anulare de 3s) | Atingere | Anulare ultimul scor |
