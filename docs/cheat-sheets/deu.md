# Kurzanleitung zur Uhrensteuerung

## Physische Tasten

| Taste | Aktion | Funktion |
|---|---|---|
| **START/STOP** (top-right) | Drücken | Aufzeichnung starten oder pausieren |
| **START/STOP** (top-right) | Drücken (im Pausenmenü) | Ausgewählte Option bestätigen (Fortsetzen/Speichern/Verwerfen) |
| **UP/MENU** (top-left) | Drücken | +1 linker Spielstand |
| **UP/MENU** (top-left) | Lang drücken | Spielstand-Korrekturmodus aktivieren (linke Seite ausgewählt) |
| **UP/MENU** (top-left) | Drücken (im Korrekturmodus) | Zwischen linker/rechter Seite wechseln |
| **DOWN** (bottom-left) | Drücken | +1 rechter Spielstand |
| **DOWN** (bottom-left) | Drücken (im Korrekturmodus) | −1 auf ausgewählter Seite |
| **DOWN** (bottom-left) | Drücken (im Bestätigungsbildschirm) | Zurück zum Pausenmenü |
| **BACK/LAP** (bottom-right) | Drücken | Torwart-Timer zurücksetzen |
| **BACK/LAP** (bottom-right) | Lang drücken | Torwart-Timer-Einstellungen öffnen |
| **BACK/LAP** (bottom-right) | Drücken (im Korrekturmodus) | Spielstand-Korrekturmodus verlassen |
| **BACK/LAP** (bottom-right) | Drücken (im Bestätigungsbildschirm) | Zurück zum Pausenmenü |

## Touchscreen-Bereiche

| Bereich | Aktion | Funktion |
|---|---|---|
| Linke Bildschirmhälfte | Antippen | +1 linker Spielstand |
| Rechte Bildschirmhälfte | Antippen | +1 rechter Spielstand |
| Untere ~30% des Bildschirms | Antippen | Torwart-Timer zurücksetzen |
| Hervorgehobene Seite (während des 3s Rückgängig-Fensters) | Antippen | Letzten Spielstand rückgängig machen |
