# Snelgids horlogebediening

## Fysieke knoppen

| Knop | Actie | Functie |
|---|---|---|
| **START/STOP** (top-right) | Drukken | Opname starten of pauzeren |
| **START/STOP** (top-right) | Drukken (in pauzemenu) | Geselecteerde optie bevestigen (Hervatten/Opslaan/Verwijderen) |
| **UP/MENU** (top-left) | Drukken | +1 linkerscore |
| **UP/MENU** (top-left) | Lang indrukken | Scorecorrectiemodus openen (linkerzijde geselecteerd) |
| **UP/MENU** (top-left) | Drukken (in scorecorrectie) | Wisselen tussen linker-/rechterzijde |
| **DOWN** (bottom-left) | Drukken | +1 rechterscore |
| **DOWN** (bottom-left) | Drukken (in scorecorrectie) | −1 op geselecteerde zijde |
| **DOWN** (bottom-left) | Drukken (in bevestigingsscherm) | Terug naar pauzemenu |
| **BACK/LAP** (bottom-right) | Drukken | Keepertimer resetten |
| **BACK/LAP** (bottom-right) | Lang indrukken | Keepertimer-instellingen openen |
| **BACK/LAP** (bottom-right) | Drukken (in scorecorrectie) | Scorecorrectiemodus verlaten |
| **BACK/LAP** (bottom-right) | Drukken (in bevestigingsscherm) | Terug naar pauzemenu |

## Touchscreenzones

| Zone | Actie | Functie |
|---|---|---|
| Linkerhelft van het scherm | Tikken | +1 linkerscore |
| Rechterhelft van het scherm | Tikken | +1 rechterscore |
| Onderste ~30% van het scherm | Tikken | Keepertimer resetten |
| Gemarkeerde zijde (tijdens 3s ongedaan-maken-venster) | Tikken | Laatste score ongedaan maken |
