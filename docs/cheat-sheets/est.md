# Kella juhtimine - spikker

## Fuusilised nupud

| Nupp | Tegevus | Mida see teeb |
|---|---|---|
| **START/STOP** (top-right) | Vajuta | Alusta v6i peata salvestamine |
| **START/STOP** (top-right) | Vajuta (pausimenUUs) | Kinnita valitud valik (Jatka/Salvesta/Loobu) |
| **UP/MENU** (top-left) | Vajuta | +1 vasak skoor |
| **UP/MENU** (top-left) | Pikk vajutus | Sisene skoori muutmise re~iimi (vasak pool valitud) |
| **UP/MENU** (top-left) | Vajuta (skoori muutmise re~iimis) | L"ulita vasaku/parema poole vahel |
| **DOWN** (bottom-left) | Vajuta | +1 parem skoor |
| **DOWN** (bottom-left) | Vajuta (skoori muutmise re~iimis) | −1 valitud poolel |
| **DOWN** (bottom-left) | Vajuta (kinnitusekraanil) | Tagasi pausimenUUsse |
| **BACK/LAP** (bottom-right) | Vajuta | L"ahesta v"aravavahi taimer |
| **BACK/LAP** (bottom-right) | Pikk vajutus | Ava v"aravavahi taimeri seaded |
| **BACK/LAP** (bottom-right) | Vajuta (skoori muutmise re~iimis) | V"alju skoori muutmise re~iimist |
| **BACK/LAP** (bottom-right) | Vajuta (kinnitusekraanil) | Tagasi pausimenUUsse |

## Puuteekraani tsoonid

| Tsoon | Tegevus | Mida see teeb |
|---|---|---|
| Ekraani vasak pool | Puuduta | +1 vasak skoor |
| Ekraani parem pool | Puuduta | +1 parem skoor |
| Ekraani alumine ~30% | Puuduta | L"ahesta v"aravavahi taimer |
| Esilet~ostetud pool (3s tagasiv~otmise akna ajal) | Puuduta | T"uhista viimane skoor |
