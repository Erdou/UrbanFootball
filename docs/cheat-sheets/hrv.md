# Kratke upute za upravljanje satom

## Fizicke tipke

| Tipka | Radnja | Sto radi |
|---|---|---|
| **START/STOP** (top-right) | Pritisnite | Pokreni ili pauziraj snimanje |
| **START/STOP** (top-right) | Pritisnite (u izborniku pauze) | Potvrdi odabranu opciju (Nastavi/Spremi/Odbaci) |
| **UP/MENU** (top-left) | Pritisnite | +1 lijevi rezultat |
| **UP/MENU** (top-left) | Dugi pritisak | Udi u nacin prilagodbe rezultata (odabrana lijeva strana) |
| **UP/MENU** (top-left) | Pritisnite (u nacinu prilagodbe) | Prebacivanje izmedu lijeve/desne strane |
| **DOWN** (bottom-left) | Pritisnite | +1 desni rezultat |
| **DOWN** (bottom-left) | Pritisnite (u nacinu prilagodbe) | −1 na odabranoj strani |
| **DOWN** (bottom-left) | Pritisnite (na zaslonu potvrde) | Natrag u izbornik pauze |
| **BACK/LAP** (bottom-right) | Pritisnite | Resetiraj mjerac golmana |
| **BACK/LAP** (bottom-right) | Dugi pritisak | Otvori postavke mjeraca golmana |
| **BACK/LAP** (bottom-right) | Pritisnite (u nacinu prilagodbe) | Izadi iz nacina prilagodbe rezultata |
| **BACK/LAP** (bottom-right) | Pritisnite (na zaslonu potvrde) | Natrag u izbornik pauze |

## Zone zaslona osjetljivog na dodir

| Zona | Radnja | Sto radi |
|---|---|---|
| Lijeva polovica zaslona | Dodirnite | +1 lijevi rezultat |
| Desna polovica zaslona | Dodirnite | +1 desni rezultat |
| Donjih ~30% zaslona | Dodirnite | Resetiraj mjerac golmana |
| Oznacena strana (tijekom 3s prozora za ponistavanje) | Dodirnite | Ponisti zadnji rezultat |
