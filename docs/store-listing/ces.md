# Football Scorekeeper

Dokonaly panel pro vase pratelske zapasy a 5-a-side utkani.

## Funkce

- **Tabule skore:** Klepnete vlevo (Tym A) nebo vpravo (Tym B) pro pridani golu.
- **Casovac brankaere:** Klepnete na spodni cast pro resetovani casovace brankaere (s vibraci).
- **Zaznam aktivity:** Sleduje GPS, tepovou frekvenci a ulozi zapas jako fotbalovou aktivitu.
- **Pouze dotykovy displej:** Optimalizovano pro Fenix 7/8, Epix, Venu a dotykove modely Forerunner.

## Ovladani

### Fyzicka tlacitka

| Tlacitko | Akce | Co dela |
|---|---|---|
| **START/STOP** (top-right) | Stisknete | Spustit nebo pozastavit zaznam |
| **START/STOP** (top-right) | Stisknete (v menu pauzy) | Potvrdit vybranou moznost (Pokracovat/Ulozit/Zahodit) |
| **UP/MENU** (top-left) | Stisknete | +1 leve skore |
| **UP/MENU** (top-left) | Dlouhy stisk | Vstoupit do rezimu upravy skore (vybrana leva strana) |
| **UP/MENU** (top-left) | Stisknete (v uprave skore) | Prepnout mezi levou/pravou stranou |
| **DOWN** (bottom-left) | Stisknete | +1 prave skore |
| **DOWN** (bottom-left) | Stisknete (v uprave skore) | −1 na vybrane strane |
| **DOWN** (bottom-left) | Stisknete (na potvrzovaci obrazovce) | Zpet do menu pauzy |
| **BACK/LAP** (bottom-right) | Stisknete | Resetovat casovac brankaere |
| **BACK/LAP** (bottom-right) | Dlouhy stisk | Otevrit konfiguraci casovace brankaere |
| **BACK/LAP** (bottom-right) | Stisknete (v uprave skore) | Opustit rezim upravy skore |
| **BACK/LAP** (bottom-right) | Stisknete (na potvrzovaci obrazovce) | Zpet do menu pauzy |

### Dotykove zony displeje

| Zona | Akce | Co dela |
|---|---|---|
| Leva polovina displeje | Klepnete | +1 leve skore |
| Prava polovina displeje | Klepnete | +1 prave skore |
| Spodnich ~30% displeje | Klepnete | Resetovat casovac brankaere |
| Zvyraznena strana (behem 3s okna pro vraceni) | Klepnete | Vratit posledni bod |
