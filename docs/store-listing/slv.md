# Football Scorekeeper

Najboljsa nadzorna plosca za vase rekreativne tekme in 5-a-side dvoboje.

## Funkcije

- **Semafor:** Tapnite levo (Ekipa A) ali desno (Ekipa B) za dodajanje golov.
- **Casovnik vratarja:** Tapnite spodnje podrocje za ponastavitev casovnika vratarja (z vibriranjem).
- **Snemanje aktivnosti:** Sledi GPS, srcni utrip in shrani tekmo kot nogometno aktivnost.
- **Samo zaslon na dotik:** Optimizirano za Fenix 7/8, Epix, Venu in modele Forerunner z zaslonom na dotik.

## Upravljanje

### Fizicni gumbi

| Gumb | Dejanje | Kaj naredi |
|---|---|---|
| **START/STOP** (top-right) | Pritisnite | Zacnite ali zaustavite snemanje |
| **START/STOP** (top-right) | Pritisnite (v meniju premora) | Potrdite izbrano moznost (Nadaljuj/Shrani/Zavrzi) |
| **UP/MENU** (top-left) | Pritisnite | +1 levi rezultat |
| **UP/MENU** (top-left) | Dolg pritisk | Vstopite v nacin prilagajanja rezultata (izbrana leva stran) |
| **UP/MENU** (top-left) | Pritisnite (v prilagajanju rezultata) | Preklapljajte med levo/desno stranjo |
| **DOWN** (bottom-left) | Pritisnite | +1 desni rezultat |
| **DOWN** (bottom-left) | Pritisnite (v prilagajanju rezultata) | −1 na izbrani strani |
| **DOWN** (bottom-left) | Pritisnite (na potrditvenem zaslonu) | Nazaj v meni premora |
| **BACK/LAP** (bottom-right) | Pritisnite | Ponastavite casovnik vratarja |
| **BACK/LAP** (bottom-right) | Dolg pritisk | Odprite konfiguracijo casovnika vratarja |
| **BACK/LAP** (bottom-right) | Pritisnite (v prilagajanju rezultata) | Zapustite nacin prilagajanja rezultata |
| **BACK/LAP** (bottom-right) | Pritisnite (na potrditvenem zaslonu) | Nazaj v meni premora |

### Cone zaslona na dotik

| Cona | Dejanje | Kaj naredi |
|---|---|---|
| Leva polovica zaslona | Tapnite | +1 levi rezultat |
| Desna polovica zaslona | Tapnite | +1 desni rezultat |
| Spodnjih ~30% zaslona | Tapnite | Ponastavite casovnik vratarja |
| Oznacena stran (med 3s oknom za razveljavitev) | Tapnite | Razveljavite zadnjo tocko |
