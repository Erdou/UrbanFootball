# Football Scorekeeper

The ultimate dashboard for your pickup games and 5-a-side matches.

## Features

- **Scoreboard:** Tap Left (Team A) or Right (Team B) to add goals.
- **Goalie Timer:** Tap the bottom area to reset the goalkeeper timer (with vibration).
- **Activity Recording:** Tracks GPS, Heart Rate, and saves the match as a Football activity.
- **Touchscreen Only:** Optimized for Fenix 7/8, Epix, Venu, and Forerunner touch models.

## Controls

### Physical Buttons

| Button | Action | What it does |
|---|---|---|
| **START/STOP** (top-right) | Press | Start or pause recording |
| **START/STOP** (top-right) | Press (in pause menu) | Confirm selected option (Resume/Save/Discard) |
| **UP/MENU** (top-left) | Press | +1 left score |
| **UP/MENU** (top-left) | Long press | Enter Score Adjust Mode (left side selected) |
| **UP/MENU** (top-left) | Press (in Score Adjust) | Toggle between left/right side |
| **DOWN** (bottom-left) | Press | +1 right score |
| **DOWN** (bottom-left) | Press (in Score Adjust) | −1 on selected side |
| **DOWN** (bottom-left) | Press (in confirm screen) | Go back to pause menu |
| **BACK/LAP** (bottom-right) | Press | Reset goalie timer |
| **BACK/LAP** (bottom-right) | Long press | Open goalie timer config |
| **BACK/LAP** (bottom-right) | Press (in Score Adjust) | Exit Score Adjust Mode |
| **BACK/LAP** (bottom-right) | Press (in confirm screen) | Go back to pause menu |

### Touchscreen Zones

| Zone | Action | What it does |
|---|---|---|
| Left half of screen | Tap | +1 left score |
| Right half of screen | Tap | +1 right score |
| Bottom ~30% of screen | Tap | Reset goalie timer |
| Highlighted side (during 3s undo window) | Tap | Undo last score |
