# Voetbal Scorekeeper

Het ultieme dashboard voor je potjes en 5-a-side wedstrijden.

## Functies

- **Scorebord:** Tik links (Team A) of rechts (Team B) om doelpunten toe te voegen.
- **Keeper-timer:** Tik op het onderste gedeelte om de keeper-timer te resetten (met trilsignaal).
- **Activiteitsregistratie:** Volgt GPS, hartslag en slaat de wedstrijd op als voetbalactiviteit.
- **Alleen touchscreen:** Geoptimaliseerd voor Fenix 7/8, Epix, Venu en Forerunner touch-modellen.

## Bediening

### Fysieke knoppen

| Knop | Actie | Wat het doet |
|---|---|---|
| **START/STOP** (top-right) | Drukken | Opname starten of pauzeren |
| **START/STOP** (top-right) | Drukken (in pauzemenu) | Geselecteerde optie bevestigen (Hervatten/Opslaan/Verwijderen) |
| **UP/MENU** (top-left) | Drukken | +1 linkerscore |
| **UP/MENU** (top-left) | Lang indrukken | Score-aanpassingsmodus openen (linkerzijde geselecteerd) |
| **UP/MENU** (top-left) | Drukken (in Score-aanpassing) | Wisselen tussen linker-/rechterzijde |
| **DOWN** (bottom-left) | Drukken | +1 rechterscore |
| **DOWN** (bottom-left) | Drukken (in Score-aanpassing) | −1 op geselecteerde zijde |
| **DOWN** (bottom-left) | Drukken (in bevestigingsscherm) | Terug naar pauzemenu |
| **BACK/LAP** (bottom-right) | Drukken | Keeper-timer resetten |
| **BACK/LAP** (bottom-right) | Lang indrukken | Keeper-timer-configuratie openen |
| **BACK/LAP** (bottom-right) | Drukken (in Score-aanpassing) | Score-aanpassingsmodus verlaten |
| **BACK/LAP** (bottom-right) | Drukken (in bevestigingsscherm) | Terug naar pauzemenu |

### Touchscreen-zones

| Zone | Actie | Wat het doet |
|---|---|---|
| Linkerhelft van het scherm | Tikken | +1 linkerscore |
| Rechterhelft van het scherm | Tikken | +1 rechterscore |
| Onderste ~30% van het scherm | Tikken | Keeper-timer resetten |
| Gemarkeerde zijde (tijdens 3s ongedaan-maken-venster) | Tikken | Laatste score ongedaan maken |
