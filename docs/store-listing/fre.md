# Compteur de Buts Football

Le tableau de bord ultime pour vos matchs improvisés et vos parties à 5-a-side.

## Fonctionnalités

- **Tableau de score :** Appuyez à gauche (Équipe A) ou à droite (Équipe B) pour ajouter des buts.
- **Chrono Gardien :** Appuyez sur la zone inférieure pour réinitialiser le chrono du gardien (avec vibration).
- **Enregistrement d'activité :** Suit le GPS, la fréquence cardiaque et enregistre le match en tant qu'activité Football.
- **Écran tactile uniquement :** Optimisé pour Fenix 7/8, Epix, Venu et les modèles tactiles Forerunner.

## Commandes

### Boutons physiques

| Bouton | Action | Fonction |
|---|---|---|
| **START/STOP** (top-right) | Appui | Démarrer ou mettre en pause l'enregistrement |
| **START/STOP** (top-right) | Appui (dans le menu pause) | Confirmer l'option sélectionnée (Reprendre/Sauvegarder/Supprimer) |
| **UP/MENU** (top-left) | Appui | +1 score gauche |
| **UP/MENU** (top-left) | Appui long | Entrer en mode Ajustement du score (côté gauche sélectionné) |
| **UP/MENU** (top-left) | Appui (en Ajustement du score) | Basculer entre côté gauche/droit |
| **DOWN** (bottom-left) | Appui | +1 score droit |
| **DOWN** (bottom-left) | Appui (en Ajustement du score) | −1 sur le côté sélectionné |
| **DOWN** (bottom-left) | Appui (sur l'écran de confirmation) | Revenir au menu pause |
| **BACK/LAP** (bottom-right) | Appui | Réinitialiser le chrono gardien |
| **BACK/LAP** (bottom-right) | Appui long | Ouvrir la configuration du chrono gardien |
| **BACK/LAP** (bottom-right) | Appui (en Ajustement du score) | Quitter le mode Ajustement du score |
| **BACK/LAP** (bottom-right) | Appui (sur l'écran de confirmation) | Revenir au menu pause |

### Zones tactiles

| Zone | Action | Fonction |
|---|---|---|
| Moitié gauche de l'écran | Appui | +1 score gauche |
| Moitié droite de l'écran | Appui | +1 score droit |
| ~30% inférieur de l'écran | Appui | Réinitialiser le chrono gardien |
| Côté surligné (pendant la fenêtre d'annulation de 3s) | Appui | Annuler le dernier score |
