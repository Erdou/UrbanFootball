# Fotball Scorekeeper

Det ultimate dashboardet for dine treningskamper og 5-a-side kamper.

## Funksjoner

- **Resultattavle:** Trykk venstre (Lag A) eller hoyre (Lag B) for a legge til mal.
- **Malvakttimer:** Trykk pa det nedre omradet for a tilbakestille malvakttimeren (med vibrasjon).
- **Aktivitetsregistrering:** Sporer GPS, puls og lagrer kampen som en fotballaktivitet.
- **Kun berøringsskjerm:** Optimalisert for Fenix 7/8, Epix, Venu og Forerunner berøringsmodeller.

## Kontroller

### Fysiske knapper

| Knapp | Handling | Hva den gjor |
|---|---|---|
| **START/STOP** (top-right) | Trykk | Start eller pause opptak |
| **START/STOP** (top-right) | Trykk (i pausemeny) | Bekreft valgt alternativ (Gjenoppta/Lagre/Forkast) |
| **UP/MENU** (top-left) | Trykk | +1 venstre poengsum |
| **UP/MENU** (top-left) | Langt trykk | Apne poengjusteringsmodus (venstre side valgt) |
| **UP/MENU** (top-left) | Trykk (i Poengjustering) | Bytt mellom venstre/hoyre side |
| **DOWN** (bottom-left) | Trykk | +1 hoyre poengsum |
| **DOWN** (bottom-left) | Trykk (i Poengjustering) | −1 pa valgt side |
| **DOWN** (bottom-left) | Trykk (i bekreftelsesskjerm) | Tilbake til pausemeny |
| **BACK/LAP** (bottom-right) | Trykk | Tilbakestill malvakttimer |
| **BACK/LAP** (bottom-right) | Langt trykk | Apne malvakttimer-konfigurasjon |
| **BACK/LAP** (bottom-right) | Trykk (i Poengjustering) | Avslutt poengjusteringsmodus |
| **BACK/LAP** (bottom-right) | Trykk (i bekreftelsesskjerm) | Tilbake til pausemeny |

### Berøringsskjerm-soner

| Sone | Handling | Hva den gjor |
|---|---|---|
| Venstre halvdel av skjermen | Trykk | +1 venstre poengsum |
| Hoyre halvdel av skjermen | Trykk | +1 hoyre poengsum |
| Nedre ~30% av skjermen | Trykk | Tilbakestill malvakttimer |
| Uthevet side (under 3s angre-vindu) | Trykk | Angre siste poengsum |
