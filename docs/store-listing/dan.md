# Fodbold Scorekeeper

Det ultimative dashboard til dine amatorkampe og 5-a-side kampe.

## Funktioner

- **Resultattavle:** Tryk venstre (Hold A) eller hojre (Hold B) for at tilfoeje mal.
- **Malvogter-timer:** Tryk pa det nederste omrade for at nulstille malvogter-timeren (med vibration).
- **Aktivitetsregistrering:** Sporer GPS, puls og gemmer kampen som en fodboldaktivitet.
- **Kun touchscreen:** Optimeret til Fenix 7/8, Epix, Venu og Forerunner touch-modeller.

## Betjening

### Fysiske knapper

| Knap | Handling | Hvad den gor |
|---|---|---|
| **START/STOP** (top-right) | Tryk | Start eller pause optagelse |
| **START/STOP** (top-right) | Tryk (i pausemenu) | Bekraeft valgt mulighed (Genoptag/Gem/Kassér) |
| **UP/MENU** (top-left) | Tryk | +1 venstre score |
| **UP/MENU** (top-left) | Langt tryk | Abn score-justeringstilstand (venstre side valgt) |
| **UP/MENU** (top-left) | Tryk (i Score-justering) | Skift mellem venstre/hojre side |
| **DOWN** (bottom-left) | Tryk | +1 hojre score |
| **DOWN** (bottom-left) | Tryk (i Score-justering) | −1 pa valgt side |
| **DOWN** (bottom-left) | Tryk (i bekraeftelsesskarm) | Tilbage til pausemenu |
| **BACK/LAP** (bottom-right) | Tryk | Nulstil malvogter-timer |
| **BACK/LAP** (bottom-right) | Langt tryk | Abn malvogter-timer-konfiguration |
| **BACK/LAP** (bottom-right) | Tryk (i Score-justering) | Forlad score-justeringstilstand |
| **BACK/LAP** (bottom-right) | Tryk (i bekraeftelsesskarm) | Tilbage til pausemenu |

### Touchscreen-zoner

| Zone | Handling | Hvad den gor |
|---|---|---|
| Venstre halvdel af skaermen | Tryk | +1 venstre score |
| Hojre halvdel af skaermen | Tryk | +1 hojre score |
| Nederste ~30% af skaermen | Tryk | Nulstil malvogter-timer |
| Fremhaevet side (under 3s fortryd-vindue) | Tryk | Fortryd sidste score |
