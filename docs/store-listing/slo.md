# Football Scorekeeper

Dokonaly panel pre vase priatelske zapasy a 5-a-side stretnutia.

## Funkcie

- **Tabula skore:** Klepnite vlavo (Tim A) alebo vpravo (Tim B) pre pridanie golu.
- **Casovac brankara:** Klepnite na spodnu cast pre resetovanie casovaca brankara (s vibraciou).
- **Zaznam aktivity:** Sleduje GPS, tepovu frekvenciu a ulozi zapas ako futbalovu aktivitu.
- **Iba dotykovy displej:** Optimalizovane pre Fenix 7/8, Epix, Venu a dotykove modely Forerunner.

## Ovladanie

### Fyzicke tlacidla

| Tlacidlo | Akcia | Co robi |
|---|---|---|
| **START/STOP** (top-right) | Stlacte | Spustit alebo pozastavit zaznam |
| **START/STOP** (top-right) | Stlacte (v menu pauzy) | Potvrdit vybranu moznost (Pokracovat/Ulozit/Zahodit) |
| **UP/MENU** (top-left) | Stlacte | +1 lave skore |
| **UP/MENU** (top-left) | Dlhe stlacenie | Vstup do rezimu upravy skore (vybrana lava strana) |
| **UP/MENU** (top-left) | Stlacte (v uprave skore) | Prepnut medzi lavou/pravou stranou |
| **DOWN** (bottom-left) | Stlacte | +1 prave skore |
| **DOWN** (bottom-left) | Stlacte (v uprave skore) | −1 na vybranej strane |
| **DOWN** (bottom-left) | Stlacte (na potvrdzovacej obrazovke) | Spat do menu pauzy |
| **BACK/LAP** (bottom-right) | Stlacte | Resetovat casovac brankara |
| **BACK/LAP** (bottom-right) | Dlhe stlacenie | Otvorit konfiguraciu casovaca brankara |
| **BACK/LAP** (bottom-right) | Stlacte (v uprave skore) | Opustit rezim upravy skore |
| **BACK/LAP** (bottom-right) | Stlacte (na potvrdzovacej obrazovke) | Spat do menu pauzy |

### Dotykove zony displeja

| Zona | Akcia | Co robi |
|---|---|---|
| Lava polovica displeja | Klepnite | +1 lave skore |
| Prava polovica displeja | Klepnite | +1 prave skore |
| Spodnych ~30% displeja | Klepnite | Resetovat casovac brankara |
| Zvyraznena strana (pocas 3s okna na vratenie) | Klepnite | Vratit posledny bod |
