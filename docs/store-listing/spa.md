# Marcador de Fútbol

El panel definitivo para tus partidos improvisados y encuentros de 5-a-side.

## Características

- **Marcador:** Toca a la izquierda (Equipo A) o a la derecha (Equipo B) para añadir goles.
- **Temporizador de Portero:** Toca la zona inferior para reiniciar el temporizador del portero (con vibración).
- **Registro de Actividad:** Registra GPS, frecuencia cardíaca y guarda el partido como actividad de Fútbol.
- **Solo pantalla táctil:** Optimizado para Fenix 7/8, Epix, Venu y modelos táctiles Forerunner.

## Controles

### Botones físicos

| Botón | Acción | Función |
|---|---|---|
| **START/STOP** (top-right) | Pulsar | Iniciar o pausar la grabación |
| **START/STOP** (top-right) | Pulsar (en menú de pausa) | Confirmar la opción seleccionada (Reanudar/Guardar/Descartar) |
| **UP/MENU** (top-left) | Pulsar | +1 puntuación izquierda |
| **UP/MENU** (top-left) | Pulsación larga | Entrar en modo Ajuste de Puntuación (lado izquierdo seleccionado) |
| **UP/MENU** (top-left) | Pulsar (en Ajuste de Puntuación) | Alternar entre lado izquierdo/derecho |
| **DOWN** (bottom-left) | Pulsar | +1 puntuación derecha |
| **DOWN** (bottom-left) | Pulsar (en Ajuste de Puntuación) | −1 en el lado seleccionado |
| **DOWN** (bottom-left) | Pulsar (en pantalla de confirmación) | Volver al menú de pausa |
| **BACK/LAP** (bottom-right) | Pulsar | Reiniciar temporizador de portero |
| **BACK/LAP** (bottom-right) | Pulsación larga | Abrir configuración del temporizador de portero |
| **BACK/LAP** (bottom-right) | Pulsar (en Ajuste de Puntuación) | Salir del modo Ajuste de Puntuación |
| **BACK/LAP** (bottom-right) | Pulsar (en pantalla de confirmación) | Volver al menú de pausa |

### Zonas táctiles

| Zona | Acción | Función |
|---|---|---|
| Mitad izquierda de la pantalla | Tocar | +1 puntuación izquierda |
| Mitad derecha de la pantalla | Tocar | +1 puntuación derecha |
| ~30% inferior de la pantalla | Tocar | Reiniciar temporizador de portero |
| Lado resaltado (durante la ventana de deshacer de 3s) | Tocar | Deshacer última puntuación |
