# Futbol Skor Takipçisi

Halı saha maçlarınız ve 5-a-side karşılaşmalarınız için mükemmel kontrol paneli.

## Özellikler

- **Skor Tahtası:** Sol (Takım A) veya Sağ (Takım B) tarafa dokunarak gol ekleyin.
- **Kaleci Sayacı:** Alt alana dokunarak kaleci sayacını sıfırlayın (titreşimli).
- **Aktivite Kaydı:** GPS ve kalp atış hızını takip eder, maçı futbol aktivitesi olarak kaydeder.
- **Yalnızca Dokunmatik Ekran:** Fenix 7/8, Epix, Venu ve dokunmatik Forerunner modelleri için optimize edilmiştir.

## Kontroller

### Fiziksel Düğmeler

| Düğme | Eylem | Ne yapar |
|---|---|---|
| **START/STOP** (top-right) | Basın | Kaydı başlat veya duraklat |
| **START/STOP** (top-right) | Basın (duraklatma menüsünde) | Seçili seçeneği onayla (Devam/Kaydet/Sil) |
| **UP/MENU** (top-left) | Basın | +1 sol skor |
| **UP/MENU** (top-left) | Uzun basın | Skor Düzenleme Moduna gir (sol taraf seçili) |
| **UP/MENU** (top-left) | Basın (Skor Düzenleme Modunda) | Sol/sağ taraf arasında geçiş yap |
| **DOWN** (bottom-left) | Basın | +1 sağ skor |
| **DOWN** (bottom-left) | Basın (Skor Düzenleme Modunda) | −1 seçili tarafta |
| **DOWN** (bottom-left) | Basın (onay ekranında) | Duraklatma menüsüne geri dön |
| **BACK/LAP** (bottom-right) | Basın | Kaleci sayacını sıfırla |
| **BACK/LAP** (bottom-right) | Uzun basın | Kaleci sayacı ayarlarını aç |
| **BACK/LAP** (bottom-right) | Basın (Skor Düzenleme Modunda) | Skor Düzenleme Modundan çık |
| **BACK/LAP** (bottom-right) | Basın (onay ekranında) | Duraklatma menüsüne geri dön |

### Dokunmatik Ekran Bölgeleri

| Bölge | Eylem | Ne yapar |
|---|---|---|
| Ekranın sol yarısı | Dokunun | +1 sol skor |
| Ekranın sağ yarısı | Dokunun | +1 sağ skor |
| Ekranın alt ~30%'u | Dokunun | Kaleci sayacını sıfırla |
| Vurgulanan taraf (3s geri alma penceresi sırasında) | Dokunun | Son skoru geri al |
