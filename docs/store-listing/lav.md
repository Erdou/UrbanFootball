# Futbola rezultatu skaitltajs

Labakais panelisvisam tavam draudzibas un 5-a-side spelem.

## Funkcijas

- **Rezultatu tabula:** Pieskaries pa kreisi (Komanda A) vai pa labi (Komanda B), lai pievienotu vartus.
- **Vartsarga taimeris:** Pieskaries apaksai, lai atiestatitu vartsarga taimeri (ar vibraciju).
- **Aktivitates ierakstisana:** Izseko GPS, sirdsdarbibas frekvenci un saglaba spelikaaktivitati.
- **Tikai skarienekrans:** Optimizets Fenix 7/8, Epix, Venu un Forerunner skarienmodeliem.

## Vadiba

### Fiziskas pogas

| Poga | Darbiba | Ko ta dara |
|---|---|---|
| **START/STOP** (top-right) | Nospiest | Sakt vai apturet ierakstisanu |
| **START/STOP** (top-right) | Nospiest (pauzes izvelne) | Apstiprinat izveletoiespieju (Turpinat/Saglabatat/Dzest) |
| **UP/MENU** (top-left) | Nospiest | +1 kreisais rezultats |
| **UP/MENU** (top-left) | Ilgs nospiedienis | Ieiet rezultatu koreksanas rezima (kreisa puse izvelta) |
| **UP/MENU** (top-left) | Nospiest (rezultatu koreksana) | Parslegt starp kreiso/labo pusi |
| **DOWN** (bottom-left) | Nospiest | +1 labais rezultats |
| **DOWN** (bottom-left) | Nospiest (rezultatu koreksana) | −1 izveleetaja puse |
| **DOWN** (bottom-left) | Nospiest (apstiprinajuma ekrana) | Atpakal uz pauzes izvelni |
| **BACK/LAP** (bottom-right) | Nospiest | Atiestatit vartsarga taimeri |
| **BACK/LAP** (bottom-right) | Ilgs nospiediens | Avert vartsarga taimera iestatijumus |
| **BACK/LAP** (bottom-right) | Nospiest (rezultatu koreksana) | Iziet no rezultatu koreksanas rezima |
| **BACK/LAP** (bottom-right) | Nospiest (apstiprinajuma ekrana) | Atpakal uz pauzes izvelni |

### Skarienekrana zonas

| Zona | Darbiba | Ko ta dara |
|---|---|---|
| Ekrana kreisa puse | Pieskaries | +1 kreisais rezultats |
| Ekrana laba puse | Pieskaries | +1 labais rezultats |
| Ekrana apaksejie ~30% | Pieskaries | Atiestatit vartsarga taimeri |
| Izcelta puse (3s atsauksanas loga laika) | Pieskaries | Atsaukt pedejo rezultatu |
