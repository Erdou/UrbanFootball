# Futbolo rezultatu skaitiklis

Geriausia prietaisu skydelis tavo draugiskoms ir 5-a-side rungtyneems.

## Funkcijos

- **Rezultatu lenta:** Bakstelekite kaire (Komanda A) arba desine (Komanda B), kad prideetumete ivarti.
- **Vartininko laikmatis:** Bakstelekite apatine zona, kad atstatytumete vartininko laikmati (su vibracija).
- **Veiklos irasymas:** Seka GPS, sirdies ritma ir issaugo rungtynes kaip futbolo veikla.
- **Tik jutiklinis ekranas:** Optimizuota Fenix 7/8, Epix, Venu ir Forerunner jutikliniams modeliams.

## Valdymas

### Fiziniai mygtukai

| Mygtukas | Veiksmas | Ka tai daro |
|---|---|---|
| **START/STOP** (top-right) | Paspauskite | Pradeti arba pristabdyti irasyma |
| **START/STOP** (top-right) | Paspauskite (pauzes meniu) | Patvirtinti pasirinkta varianta (Testi/Issaugoti/Atmesti) |
| **UP/MENU** (top-left) | Paspauskite | +1 kairysis rezultatas |
| **UP/MENU** (top-left) | Ilgas paspaudimas | Ieiti i rezultatu koregavimo rezima (kaire puse pasirinkta) |
| **UP/MENU** (top-left) | Paspauskite (rezultatu koregavime) | Perjungti tarp kaires/desines puses |
| **DOWN** (bottom-left) | Paspauskite | +1 desinysis rezultatas |
| **DOWN** (bottom-left) | Paspauskite (rezultatu koregavime) | −1 pasirinktoje puseje |
| **DOWN** (bottom-left) | Paspauskite (patvirtinimo ekrane) | Grizti i pauzes meniu |
| **BACK/LAP** (bottom-right) | Paspauskite | Atstatyti vartininko laikmati |
| **BACK/LAP** (bottom-right) | Ilgas paspaudimas | Atidaryti vartininko laikmacio nustatymus |
| **BACK/LAP** (bottom-right) | Paspauskite (rezultatu koregavime) | Iseiti is rezultatu koregavimo rezimo |
| **BACK/LAP** (bottom-right) | Paspauskite (patvirtinimo ekrane) | Grizti i pauzes meniu |

### Jutiklinio ekrano zonos

| Zona | Veiksmas | Ka tai daro |
|---|---|---|
| Kaire ekrano puse | Bakstelekite | +1 kairysis rezultatas |
| Desine ekrano puse | Bakstelekite | +1 desinysis rezultatas |
| Apatiniai ~30% ekrano | Bakstelekite | Atstatyti vartininko laikmati |
| Pazymeta puse (3s atsaukimo lango metu) | Bakstelekite | Atsaukti paskutini rezultata |
