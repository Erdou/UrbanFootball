# Pencatat Skor Bola Sepak

Papan pemuka terbaik untuk permainan bola sepak kasual dan perlawanan 5-a-side anda.

## Ciri-ciri

- **Papan Skor:** Ketik Kiri (Pasukan A) atau Kanan (Pasukan B) untuk menambah gol.
- **Pemasa Penjaga Gol:** Ketik kawasan bawah untuk menetapkan semula pemasa penjaga gol (dengan getaran).
- **Rakaman Aktiviti:** Menjejak GPS, Kadar Denyutan Jantung dan menyimpan perlawanan sebagai aktiviti Bola Sepak.
- **Skrin Sentuh Sahaja:** Dioptimumkan untuk Fenix 7/8, Epix, Venu dan model sentuh Forerunner.

## Kawalan

### Butang Fizikal

| Butang | Tindakan | Fungsi |
|---|---|---|
| **START/STOP** (top-right) | Tekan | Mula atau jeda rakaman |
| **START/STOP** (top-right) | Tekan (dalam menu jeda) | Sahkan pilihan yang dipilih (Sambung/Simpan/Buang) |
| **UP/MENU** (top-left) | Tekan | Skor kiri +1 |
| **UP/MENU** (top-left) | Tekan lama | Masuk Mod Laras Skor (bahagian kiri dipilih) |
| **UP/MENU** (top-left) | Tekan (dalam Laras Skor) | Togol antara bahagian kiri/kanan |
| **DOWN** (bottom-left) | Tekan | Skor kanan +1 |
| **DOWN** (bottom-left) | Tekan (dalam Laras Skor) | Skor bahagian yang dipilih −1 |
| **DOWN** (bottom-left) | Tekan (dalam skrin pengesahan) | Kembali ke menu jeda |
| **BACK/LAP** (bottom-right) | Tekan | Tetapkan semula pemasa penjaga gol |
| **BACK/LAP** (bottom-right) | Tekan lama | Buka konfigurasi pemasa penjaga gol |
| **BACK/LAP** (bottom-right) | Tekan (dalam Laras Skor) | Keluar Mod Laras Skor |
| **BACK/LAP** (bottom-right) | Tekan (dalam skrin pengesahan) | Kembali ke menu jeda |

### Zon Skrin Sentuh

| Zon | Tindakan | Fungsi |
|---|---|---|
| Separuh kiri skrin | Ketik | Skor kiri +1 |
| Separuh kanan skrin | Ketik | Skor kanan +1 |
| Bahagian bawah ~30% skrin | Ketik | Tetapkan semula pemasa penjaga gol |
| Bahagian yang diserlahkan (semasa tetingkap buat asal 3s) | Ketik | Buat asal skor terakhir |
