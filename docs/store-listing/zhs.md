# 足球记分员

适用于街头足球和5-a-side比赛的终极仪表盘。

## 功能

- **记分板:** 点击左侧（A队）或右侧（B队）添加进球。
- **守门员计时器:** 点击底部区域重置守门员计时器（带振动）。
- **活动记录:** 追踪GPS、心率，并将比赛保存为足球活动。
- **仅限触屏:** 针对Fenix 7/8、Epix、Venu和Forerunner触屏型号优化。

## 操作方式

### 物理按钮

| 按钮 | 操作 | 功能说明 |
|---|---|---|
| **START/STOP** (top-right) | 按下 | 开始或暂停记录 |
| **START/STOP** (top-right) | 按下（暂停菜单中） | 确认所选选项（恢复/保存/丢弃） |
| **UP/MENU** (top-left) | 按下 | 左侧得分 +1 |
| **UP/MENU** (top-left) | 长按 | 进入得分调整模式（选中左侧） |
| **UP/MENU** (top-left) | 按下（得分调整中） | 切换左侧/右侧 |
| **DOWN** (bottom-left) | 按下 | 右侧得分 +1 |
| **DOWN** (bottom-left) | 按下（得分调整中） | 所选侧得分 −1 |
| **DOWN** (bottom-left) | 按下（确认画面中） | 返回暂停菜单 |
| **BACK/LAP** (bottom-right) | 按下 | 重置守门员计时器 |
| **BACK/LAP** (bottom-right) | 长按 | 打开守门员计时器设置 |
| **BACK/LAP** (bottom-right) | 按下（得分调整中） | 退出得分调整模式 |
| **BACK/LAP** (bottom-right) | 按下（确认画面中） | 返回暂停菜单 |

### 触屏区域

| 区域 | 操作 | 功能说明 |
|---|---|---|
| 屏幕左半部分 | 点击 | 左侧得分 +1 |
| 屏幕右半部分 | 点击 | 右侧得分 +1 |
| 屏幕底部 ~30% | 点击 | 重置守门员计时器 |
| 高亮侧（3s 撤销窗口期间） | 点击 | 撤销上一次得分 |
