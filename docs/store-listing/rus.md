# Football Scorekeeper

Лучшая панель для ваших дворовых матчей и игр 5-a-side.

## Функции

- **Табло:** Нажмите слева (Команда A) или справа (Команда B), чтобы добавить гол.
- **Таймер вратаря:** Нажмите на нижнюю область, чтобы сбросить таймер вратаря (с вибрацией).
- **Запись активности:** Отслеживает GPS, пульс и сохраняет матч как футбольную активность.
- **Только сенсорный экран:** Оптимизировано для Fenix 7/8, Epix, Venu и сенсорных моделей Forerunner.

## Управление

### Физические кнопки

| Кнопка | Действие | Что делает |
|---|---|---|
| **START/STOP** (top-right) | Нажмите | Начать или приостановить запись |
| **START/STOP** (top-right) | Нажмите (в меню паузы) | Подтвердить выбранный вариант (Продолжить/Сохранить/Отменить) |
| **UP/MENU** (top-left) | Нажмите | +1 левый счёт |
| **UP/MENU** (top-left) | Долгое нажатие | Войти в режим корректировки счёта (выбрана левая сторона) |
| **UP/MENU** (top-left) | Нажмите (в корректировке счёта) | Переключение между левой/правой стороной |
| **DOWN** (bottom-left) | Нажмите | +1 правый счёт |
| **DOWN** (bottom-left) | Нажмите (в корректировке счёта) | −1 на выбранной стороне |
| **DOWN** (bottom-left) | Нажмите (на экране подтверждения) | Назад в меню паузы |
| **BACK/LAP** (bottom-right) | Нажмите | Сбросить таймер вратаря |
| **BACK/LAP** (bottom-right) | Долгое нажатие | Открыть настройки таймера вратаря |
| **BACK/LAP** (bottom-right) | Нажмите (в корректировке счёта) | Выйти из режима корректировки счёта |
| **BACK/LAP** (bottom-right) | Нажмите (на экране подтверждения) | Назад в меню паузы |

### Зоны сенсорного экрана

| Зона | Действие | Что делает |
|---|---|---|
| Левая половина экрана | Нажмите | +1 левый счёт |
| Правая половина экрана | Нажмите | +1 правый счёт |
| Нижние ~30% экрана | Нажмите | Сбросить таймер вратаря |
| Выделенная сторона (во время 3s окна отмены) | Нажмите | Отменить последнее очко |
