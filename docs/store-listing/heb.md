# מונה תוצאות כדורגל

לוח המחוונים האולטימטיבי למשחקים חופשיים ומשחקי 5-a-side.

## תכונות

- **לוח תוצאות:** הקש שמאלה (קבוצה א) או ימינה (קבוצה ב) להוספת שערים.
- **טיימר שוער:** הקש על האזור התחתון לאיפוס טיימר השוער (עם רטט).
- **הקלטת פעילות:** עוקב אחר GPS, דופק, ושומר את המשחק כפעילות כדורגל.
- **מסך מגע בלבד:** מותאם לדגמי Fenix 7/8, Epix, Venu ו-Forerunner עם מסך מגע.

## בקרות

### כפתורים פיזיים

| כפתור | פעולה | מה זה עושה |
|---|---|---|
| **START/STOP** (top-right) | לחיצה | התחלה או השהיית הקלטה |
| **START/STOP** (top-right) | לחיצה (בתפריט השהיה) | אישור האפשרות שנבחרה (המשך/שמירה/מחיקה) |
| **UP/MENU** (top-left) | לחיצה | +1 תוצאה שמאל |
| **UP/MENU** (top-left) | לחיצה ארוכה | כניסה למצב עריכת תוצאה (צד שמאל נבחר) |
| **UP/MENU** (top-left) | לחיצה (במצב עריכת תוצאה) | מעבר בין צד שמאל/ימין |
| **DOWN** (bottom-left) | לחיצה | +1 תוצאה ימין |
| **DOWN** (bottom-left) | לחיצה (במצב עריכת תוצאה) | −1 בצד שנבחר |
| **DOWN** (bottom-left) | לחיצה (במסך אישור) | חזרה לתפריט ההשהיה |
| **BACK/LAP** (bottom-right) | לחיצה | איפוס טיימר השוער |
| **BACK/LAP** (bottom-right) | לחיצה ארוכה | פתיחת הגדרות טיימר השוער |
| **BACK/LAP** (bottom-right) | לחיצה (במצב עריכת תוצאה) | יציאה ממצב עריכת תוצאה |
| **BACK/LAP** (bottom-right) | לחיצה (במסך אישור) | חזרה לתפריט ההשהיה |

### אזורי מגע במסך

| אזור | פעולה | מה זה עושה |
|---|---|---|
| חצי שמאל של המסך | הקשה | +1 תוצאה שמאל |
| חצי ימין של המסך | הקשה | +1 תוצאה ימין |
| ~30% התחתון של המסך | הקשה | איפוס טיימר השוער |
| הצד המודגש (במהלך חלון ביטול 3s) | הקשה | ביטול השער האחרון |
