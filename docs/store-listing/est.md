# Jalgpalli tulemuste loendur

Parim juhtpaneel sinu vabaaja- ja 5-a-side mängudeks.

## Funktsioonid

- **Tulemustabel:** Puuduta vasakut (Meeskond A) voi paremat (Meeskond B), et lisada varat.
- **Varavavahi taimer:** Puuduta alumist ala, et lahtestada varavavahi taimer (vibratsiooniga).
- **Tegevuse salvestamine:** Jalgib GPS-i, pulssi ja salvestab matsi kui jalgpallitegevuse.
- **Ainult puuteekraan:** Optimeeritud Fenix 7/8, Epix, Venu ja Forerunner puutemudelitele.

## Juhtimine

### Fuusilised nupud

| Nupp | Toiming | Mida teeb |
|---|---|---|
| **START/STOP** (top-right) | Vajuta | Alusta voi peata salvestamine |
| **START/STOP** (top-right) | Vajuta (pausi menuus) | Kinnita valitud valik (Jatka/Salvesta/Kustuta) |
| **UP/MENU** (top-left) | Vajuta | +1 vasak skoor |
| **UP/MENU** (top-left) | Pikk vajutus | Sisene skoori kohandamise reziiimi (vasak pool valitud) |
| **UP/MENU** (top-left) | Vajuta (skoori kohandamises) | Vaheta vasaku/parema poole vahel |
| **DOWN** (bottom-left) | Vajuta | +1 parem skoor |
| **DOWN** (bottom-left) | Vajuta (skoori kohandamises) | −1 valitud poolel |
| **DOWN** (bottom-left) | Vajuta (kinnitusekraanil) | Tagasi pausi menuusse |
| **BACK/LAP** (bottom-right) | Vajuta | Lahtesta varavavahi taimer |
| **BACK/LAP** (bottom-right) | Pikk vajutus | Ava varavavahi taimeri seaded |
| **BACK/LAP** (bottom-right) | Vajuta (skoori kohandamises) | Valju skoori kohandamise reziimist |
| **BACK/LAP** (bottom-right) | Vajuta (kinnitusekraanil) | Tagasi pausi menuusse |

### Puuteekraani tsoonid

| Tsoon | Toiming | Mida teeb |
|---|---|---|
| Ekraani vasak pool | Puuduta | +1 vasak skoor |
| Ekraani parem pool | Puuduta | +1 parem skoor |
| Ekraani alumine ~30% | Puuduta | Lahtesta varavavahi taimer |
| Esiletostud pool (3s tagasivotmise akna jooksul) | Puuduta | Tuhista viimane skoor |
