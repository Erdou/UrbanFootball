# Pencatat Skor Sepak Bola

Dasbor terbaik untuk permainan sepak bola kasual dan pertandingan 5-a-side Anda.

## Fitur

- **Papan Skor:** Ketuk Kiri (Tim A) atau Kanan (Tim B) untuk menambah gol.
- **Timer Penjaga Gawang:** Ketuk area bawah untuk mereset timer penjaga gawang (dengan getaran).
- **Perekaman Aktivitas:** Melacak GPS, Detak Jantung, dan menyimpan pertandingan sebagai aktivitas Sepak Bola.
- **Layar Sentuh Saja:** Dioptimalkan untuk Fenix 7/8, Epix, Venu, dan model sentuh Forerunner.

## Kontrol

### Tombol Fisik

| Tombol | Tindakan | Fungsi |
|---|---|---|
| **START/STOP** (top-right) | Tekan | Mulai atau jeda perekaman |
| **START/STOP** (top-right) | Tekan (di menu jeda) | Konfirmasi opsi yang dipilih (Lanjutkan/Simpan/Buang) |
| **UP/MENU** (top-left) | Tekan | Skor kiri +1 |
| **UP/MENU** (top-left) | Tekan lama | Masuk Mode Sesuaikan Skor (sisi kiri dipilih) |
| **UP/MENU** (top-left) | Tekan (di Mode Sesuaikan Skor) | Beralih antara sisi kiri/kanan |
| **DOWN** (bottom-left) | Tekan | Skor kanan +1 |
| **DOWN** (bottom-left) | Tekan (di Mode Sesuaikan Skor) | Skor sisi yang dipilih −1 |
| **DOWN** (bottom-left) | Tekan (di layar konfirmasi) | Kembali ke menu jeda |
| **BACK/LAP** (bottom-right) | Tekan | Reset timer penjaga gawang |
| **BACK/LAP** (bottom-right) | Tekan lama | Buka pengaturan timer penjaga gawang |
| **BACK/LAP** (bottom-right) | Tekan (di Mode Sesuaikan Skor) | Keluar Mode Sesuaikan Skor |
| **BACK/LAP** (bottom-right) | Tekan (di layar konfirmasi) | Kembali ke menu jeda |

### Zona Layar Sentuh

| Zona | Tindakan | Fungsi |
|---|---|---|
| Setengah kiri layar | Ketuk | Skor kiri +1 |
| Setengah kanan layar | Ketuk | Skor kanan +1 |
| Bagian bawah ~30% layar | Ketuk | Reset timer penjaga gawang |
| Sisi yang disorot (selama jendela batalkan 3s) | Ketuk | Batalkan skor terakhir |
