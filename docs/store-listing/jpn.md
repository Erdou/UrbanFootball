# フットボール スコアキーパー

ピックアップゲームや5-a-sideの試合に最適なダッシュボード。

## 機能

- **スコアボード:** 左（チームA）または右（チームB）をタップしてゴールを追加。
- **ゴールキーパータイマー:** 画面下部をタップしてゴールキーパータイマーをリセット（バイブレーション付き）。
- **アクティビティ記録:** GPS、心拍数を追跡し、試合をフットボールアクティビティとして保存。
- **タッチスクリーン専用:** Fenix 7/8、Epix、Venu、Forerunnerのタッチモデルに最適化。

## 操作方法

### 物理ボタン

| ボタン | アクション | 動作内容 |
|---|---|---|
| **START/STOP** (top-right) | 押す | 記録の開始または一時停止 |
| **START/STOP** (top-right) | 押す（一時停止メニュー内） | 選択したオプションを確定（再開/保存/破棄） |
| **UP/MENU** (top-left) | 押す | 左スコア +1 |
| **UP/MENU** (top-left) | 長押し | スコア調整モードに入る（左側を選択） |
| **UP/MENU** (top-left) | 押す（スコア調整中） | 左側/右側を切り替え |
| **DOWN** (bottom-left) | 押す | 右スコア +1 |
| **DOWN** (bottom-left) | 押す（スコア調整中） | 選択した側のスコア −1 |
| **DOWN** (bottom-left) | 押す（確認画面内） | 一時停止メニューに戻る |
| **BACK/LAP** (bottom-right) | 押す | ゴールキーパータイマーをリセット |
| **BACK/LAP** (bottom-right) | 長押し | ゴールキーパータイマー設定を開く |
| **BACK/LAP** (bottom-right) | 押す（スコア調整中） | スコア調整モードを終了 |
| **BACK/LAP** (bottom-right) | 押す（確認画面内） | 一時停止メニューに戻る |

### タッチスクリーンゾーン

| ゾーン | アクション | 動作内容 |
|---|---|---|
| 画面の左半分 | タップ | 左スコア +1 |
| 画面の右半分 | タップ | 右スコア +1 |
| 画面下部 ~30% | タップ | ゴールキーパータイマーをリセット |
| ハイライトされた側（3s 取り消しウィンドウ中） | タップ | 最後のスコアを取り消し |
