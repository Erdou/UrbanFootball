# Fussball Scorekeeper

Das ultimative Dashboard fuer deine Bolzplatz- und 5-a-side-Spiele.

## Funktionen

- **Anzeigetafel:** Tippe links (Team A) oder rechts (Team B), um Tore hinzuzufuegen.
- **Torwart-Timer:** Tippe auf den unteren Bereich, um den Torwart-Timer zurueckzusetzen (mit Vibration).
- **Aktivitaetsaufzeichnung:** Erfasst GPS, Herzfrequenz und speichert das Spiel als Fussball-Aktivitaet.
- **Nur Touchscreen:** Optimiert fuer Fenix 7/8, Epix, Venu und Forerunner Touch-Modelle.

## Steuerung

### Physische Tasten

| Taste | Aktion | Beschreibung |
|---|---|---|
| **START/STOP** (top-right) | Druecken | Aufzeichnung starten oder pausieren |
| **START/STOP** (top-right) | Druecken (im Pausenmenue) | Ausgewaehlte Option bestaetigen (Fortsetzen/Speichern/Verwerfen) |
| **UP/MENU** (top-left) | Druecken | +1 linker Punktestand |
| **UP/MENU** (top-left) | Lange druecken | Punktestand-Anpassungsmodus aufrufen (linke Seite ausgewaehlt) |
| **UP/MENU** (top-left) | Druecken (in Punktestand-Anpassung) | Zwischen linker/rechter Seite wechseln |
| **DOWN** (bottom-left) | Druecken | +1 rechter Punktestand |
| **DOWN** (bottom-left) | Druecken (in Punktestand-Anpassung) | −1 auf ausgewaehlter Seite |
| **DOWN** (bottom-left) | Druecken (im Bestaetigungsbildschirm) | Zurueck zum Pausenmenue |
| **BACK/LAP** (bottom-right) | Druecken | Torwart-Timer zuruecksetzen |
| **BACK/LAP** (bottom-right) | Lange druecken | Torwart-Timer-Konfiguration oeffnen |
| **BACK/LAP** (bottom-right) | Druecken (in Punktestand-Anpassung) | Punktestand-Anpassungsmodus verlassen |
| **BACK/LAP** (bottom-right) | Druecken (im Bestaetigungsbildschirm) | Zurueck zum Pausenmenue |

### Touchscreen-Bereiche

| Bereich | Aktion | Beschreibung |
|---|---|---|
| Linke Bildschirmhaelfte | Tippen | +1 linker Punktestand |
| Rechte Bildschirmhaelfte | Tippen | +1 rechter Punktestand |
| Untere ~30% des Bildschirms | Tippen | Torwart-Timer zuruecksetzen |
| Hervorgehobene Seite (waehrend des 3s Rueckgaengig-Fensters) | Tippen | Letzten Punktestand rueckgaengig machen |
