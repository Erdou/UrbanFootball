# Πίνακας Σκορ Ποδοσφαίρου

Ο απόλυτος πίνακας ελέγχου για τα φιλικά σας παιχνίδια και τους αγώνες 5-a-side.

## Χαρακτηριστικά

- **Πίνακας Σκορ:** Πατήστε αριστερά (Ομάδα Α) ή δεξιά (Ομάδα Β) για να προσθέσετε γκολ.
- **Χρονόμετρο Τερματοφύλακα:** Πατήστε στην κάτω περιοχή για επαναφορά του χρονομέτρου τερματοφύλακα (με δόνηση).
- **Καταγραφή Δραστηριότητας:** Παρακολουθεί GPS, καρδιακό ρυθμό και αποθηκεύει τον αγώνα ως δραστηριότητα ποδοσφαίρου.
- **Μόνο Οθόνη Αφής:** Βελτιστοποιημένο για Fenix 7/8, Epix, Venu και μοντέλα Forerunner με οθόνη αφής.

## Χειρισμοί

### Φυσικά Κουμπιά

| Κουμπί | Ενέργεια | Τι κάνει |
|---|---|---|
| **START/STOP** (top-right) | Πάτημα | Έναρξη ή παύση εγγραφής |
| **START/STOP** (top-right) | Πάτημα (στο μενού παύσης) | Επιβεβαίωση επιλεγμένης επιλογής (Συνέχεια/Αποθήκευση/Απόρριψη) |
| **UP/MENU** (top-left) | Πάτημα | +1 σκορ αριστερά |
| **UP/MENU** (top-left) | Παρατεταμένο πάτημα | Είσοδος στη Λειτουργία Ρύθμισης Σκορ (αριστερή πλευρά επιλεγμένη) |
| **UP/MENU** (top-left) | Πάτημα (στη Ρύθμιση Σκορ) | Εναλλαγή μεταξύ αριστερής/δεξιάς πλευράς |
| **DOWN** (bottom-left) | Πάτημα | +1 σκορ δεξιά |
| **DOWN** (bottom-left) | Πάτημα (στη Ρύθμιση Σκορ) | −1 στην επιλεγμένη πλευρά |
| **DOWN** (bottom-left) | Πάτημα (στην οθόνη επιβεβαίωσης) | Επιστροφή στο μενού παύσης |
| **BACK/LAP** (bottom-right) | Πάτημα | Επαναφορά χρονομέτρου τερματοφύλακα |
| **BACK/LAP** (bottom-right) | Παρατεταμένο πάτημα | Άνοιγμα ρυθμίσεων χρονομέτρου τερματοφύλακα |
| **BACK/LAP** (bottom-right) | Πάτημα (στη Ρύθμιση Σκορ) | Έξοδος από τη Λειτουργία Ρύθμισης Σκορ |
| **BACK/LAP** (bottom-right) | Πάτημα (στην οθόνη επιβεβαίωσης) | Επιστροφή στο μενού παύσης |

### Ζώνες Αφής Οθόνης

| Ζώνη | Ενέργεια | Τι κάνει |
|---|---|---|
| Αριστερό μισό της οθόνης | Πάτημα | +1 σκορ αριστερά |
| Δεξί μισό της οθόνης | Πάτημα | +1 σκορ δεξιά |
| ~30% κάτω μέρος της οθόνης | Πάτημα | Επαναφορά χρονομέτρου τερματοφύλακα |
| Επισημασμένη πλευρά (κατά τη διάρκεια παραθύρου αναίρεσης 3s) | Πάτημα | Αναίρεση τελευταίου σκορ |
