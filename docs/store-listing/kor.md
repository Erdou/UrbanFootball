# 풋볼 스코어키퍼

픽업 게임과 5-a-side 경기를 위한 최고의 대시보드.

## 기능

- **스코어보드:** 왼쪽(팀 A) 또는 오른쪽(팀 B)을 탭하여 골 추가.
- **골키퍼 타이머:** 하단 영역을 탭하여 골키퍼 타이머 리셋 (진동 포함).
- **활동 기록:** GPS, 심박수를 추적하고 경기를 풋볼 활동으로 저장.
- **터치스크린 전용:** Fenix 7/8, Epix, Venu, Forerunner 터치 모델에 최적화.

## 조작 방법

### 물리 버튼

| 버튼 | 동작 | 기능 |
|---|---|---|
| **START/STOP** (top-right) | 누르기 | 기록 시작 또는 일시정지 |
| **START/STOP** (top-right) | 누르기 (일시정지 메뉴에서) | 선택한 옵션 확인 (재개/저장/삭제) |
| **UP/MENU** (top-left) | 누르기 | 왼쪽 점수 +1 |
| **UP/MENU** (top-left) | 길게 누르기 | 점수 조정 모드 진입 (왼쪽 선택) |
| **UP/MENU** (top-left) | 누르기 (점수 조정 중) | 왼쪽/오른쪽 전환 |
| **DOWN** (bottom-left) | 누르기 | 오른쪽 점수 +1 |
| **DOWN** (bottom-left) | 누르기 (점수 조정 중) | 선택한 쪽 점수 −1 |
| **DOWN** (bottom-left) | 누르기 (확인 화면에서) | 일시정지 메뉴로 돌아가기 |
| **BACK/LAP** (bottom-right) | 누르기 | 골키퍼 타이머 리셋 |
| **BACK/LAP** (bottom-right) | 길게 누르기 | 골키퍼 타이머 설정 열기 |
| **BACK/LAP** (bottom-right) | 누르기 (점수 조정 중) | 점수 조정 모드 종료 |
| **BACK/LAP** (bottom-right) | 누르기 (확인 화면에서) | 일시정지 메뉴로 돌아가기 |

### 터치스크린 영역

| 영역 | 동작 | 기능 |
|---|---|---|
| 화면 왼쪽 절반 | 탭 | 왼쪽 점수 +1 |
| 화면 오른쪽 절반 | 탭 | 오른쪽 점수 +1 |
| 화면 하단 ~30% | 탭 | 골키퍼 타이머 리셋 |
| 강조된 쪽 (3s 실행취소 창 동안) | 탭 | 마지막 점수 실행취소 |
