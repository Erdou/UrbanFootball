# Fotboll Scorekeeper

Den ultimata instrumentpanelen for dina knackematcher och 5-a-side matcher.

## Funktioner

- **Resultattavla:** Tryck vanster (Lag A) eller hoger (Lag B) for att lagga till mal.
- **Malvaktstimer:** Tryck pa det nedre omradet for att aterstalla malvaktstimern (med vibration).
- **Aktivitetsinspelning:** Sparar GPS, puls och sparar matchen som en fotbollsaktivitet.
- **Endast pekskarm:** Optimerad for Fenix 7/8, Epix, Venu och Forerunner pekskarmsmodeller.

## Kontroller

### Fysiska knappar

| Knapp | Atgard | Vad den gor |
|---|---|---|
| **START/STOP** (top-right) | Tryck | Starta eller pausa inspelning |
| **START/STOP** (top-right) | Tryck (i pausmeny) | Bekrafta valt alternativ (Aterga/Spara/Kassera) |
| **UP/MENU** (top-left) | Tryck | +1 vanster poang |
| **UP/MENU** (top-left) | Langt tryck | Oppna poangkorrigeringslagt (vanster sida vald) |
| **UP/MENU** (top-left) | Tryck (i Poangkorrigering) | Vaxla mellan vanster/hoger sida |
| **DOWN** (bottom-left) | Tryck | +1 hoger poang |
| **DOWN** (bottom-left) | Tryck (i Poangkorrigering) | −1 pa vald sida |
| **DOWN** (bottom-left) | Tryck (i bekraftelseskarm) | Tillbaka till pausmeny |
| **BACK/LAP** (bottom-right) | Tryck | Aterstall malvaktstimer |
| **BACK/LAP** (bottom-right) | Langt tryck | Oppna malvaktstimer-konfiguration |
| **BACK/LAP** (bottom-right) | Tryck (i Poangkorrigering) | Lamna poangkorrigeringslagt |
| **BACK/LAP** (bottom-right) | Tryck (i bekraftelseskarm) | Tillbaka till pausmeny |

### Pekskarmszoner

| Zon | Atgard | Vad den gor |
|---|---|---|
| Vanster halva av skarmen | Tryck | +1 vanster poang |
| Hoger halva av skarmen | Tryck | +1 hoger poang |
| Nedre ~30% av skarmen | Tryck | Aterstall malvaktstimer |
| Markerad sida (under 3s angra-fonster) | Tryck | Angra senaste poang |
