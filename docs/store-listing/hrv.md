# Football Scorekeeper

Najbolja nadzorna ploca za vase rekreativne utakmice i 5-a-side mецeve.

## Znacajke

- **Semafor:** Dodirnite lijevo (Tim A) ili desno (Tim B) za dodavanje golova.
- **Mjerac vremena vratara:** Dodirnite donje podrucje za resetiranje mjeraca vremena vratara (s vibracijom).
- **Snimanje aktivnosti:** Prati GPS, puls i sprema utakmicu kao nogometnu aktivnost.
- **Samo zaslon na dodir:** Optimizirano za Fenix 7/8, Epix, Venu i Forerunner modele s zaslonom na dodir.

## Upravljanje

### Fizicki gumbi

| Gumb | Radnja | Sto radi |
|---|---|---|
| **START/STOP** (top-right) | Pritisnite | Pokrenite ili pauzirajte snimanje |
| **START/STOP** (top-right) | Pritisnite (u izborniku pauze) | Potvrdite odabranu opciju (Nastavi/Spremi/Odbaci) |
| **UP/MENU** (top-left) | Pritisnite | +1 lijevi rezultat |
| **UP/MENU** (top-left) | Dugi pritisak | Udite u nacin prilagodbe rezultata (odabrana lijeva strana) |
| **UP/MENU** (top-left) | Pritisnite (u prilagodbi rezultata) | Prebacujte se izmedu lijeve/desne strane |
| **DOWN** (bottom-left) | Pritisnite | +1 desni rezultat |
| **DOWN** (bottom-left) | Pritisnite (u prilagodbi rezultata) | −1 na odabranoj strani |
| **DOWN** (bottom-left) | Pritisnite (na zaslonu potvrde) | Natrag u izbornik pauze |
| **BACK/LAP** (bottom-right) | Pritisnite | Resetirajte mjerac vremena vratara |
| **BACK/LAP** (bottom-right) | Dugi pritisak | Otvorite konfiguraciju mjeraca vremena vratara |
| **BACK/LAP** (bottom-right) | Pritisnite (u prilagodbi rezultata) | Izadite iz nacina prilagodbe rezultata |
| **BACK/LAP** (bottom-right) | Pritisnite (na zaslonu potvrde) | Natrag u izbornik pauze |

### Zone zaslona na dodir

| Zona | Radnja | Sto radi |
|---|---|---|
| Lijeva polovica zaslona | Dodirnite | +1 lijevi rezultat |
| Desna polovica zaslona | Dodirnite | +1 desni rezultat |
| Donjih ~30% zaslona | Dodirnite | Resetirajte mjerac vremena vratara |
| Oznacena strana (tijekom 3s prozora za ponistenje) | Dodirnite | Ponistite zadnji bod |
