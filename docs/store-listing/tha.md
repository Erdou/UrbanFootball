# ผู้บันทึกคะแนนฟุตบอล

แดชบอร์ดที่ดีที่สุดสำหรับเกมฟุตบอลแบบจับกลุ่มและแมตช์ 5-a-side

## คุณสมบัติ

- **กระดานคะแนน:** แตะซ้าย (ทีม A) หรือขวา (ทีม B) เพื่อเพิ่มประตู
- **ตัวจับเวลาผู้รักษาประตู:** แตะพื้นที่ด้านล่างเพื่อรีเซ็ตตัวจับเวลาผู้รักษาประตู (พร้อมการสั่น)
- **บันทึกกิจกรรม:** ติดตาม GPS, อัตราการเต้นของหัวใจ และบันทึกการแข่งขันเป็นกิจกรรมฟุตบอล
- **หน้าจอสัมผัสเท่านั้น:** ปรับให้เหมาะสมสำหรับ Fenix 7/8, Epix, Venu และ Forerunner รุ่นระบบสัมผัส

## การควบคุม

### ปุ่มกด

| ปุ่ม | การกระทำ | สิ่งที่ทำ |
|---|---|---|
| **START/STOP** (top-right) | กด | เริ่มหรือหยุดชั่วคราวการบันทึก |
| **START/STOP** (top-right) | กด (ในเมนูหยุดชั่วคราว) | ยืนยันตัวเลือกที่เลือก (ดำเนินต่อ/บันทึก/ยกเลิก) |
| **UP/MENU** (top-left) | กด | คะแนนซ้าย +1 |
| **UP/MENU** (top-left) | กดค้าง | เข้าสู่โหมดปรับคะแนน (เลือกฝั่งซ้าย) |
| **UP/MENU** (top-left) | กด (ในโหมดปรับคะแนน) | สลับระหว่างฝั่งซ้าย/ขวา |
| **DOWN** (bottom-left) | กด | คะแนนขวา +1 |
| **DOWN** (bottom-left) | กด (ในโหมดปรับคะแนน) | คะแนนฝั่งที่เลือก −1 |
| **DOWN** (bottom-left) | กด (ในหน้าจอยืนยัน) | กลับไปที่เมนูหยุดชั่วคราว |
| **BACK/LAP** (bottom-right) | กด | รีเซ็ตตัวจับเวลาผู้รักษาประตู |
| **BACK/LAP** (bottom-right) | กดค้าง | เปิดการตั้งค่าตัวจับเวลาผู้รักษาประตู |
| **BACK/LAP** (bottom-right) | กด (ในโหมดปรับคะแนน) | ออกจากโหมดปรับคะแนน |
| **BACK/LAP** (bottom-right) | กด (ในหน้าจอยืนยัน) | กลับไปที่เมนูหยุดชั่วคราว |

### โซนหน้าจอสัมผัส

| โซน | การกระทำ | สิ่งที่ทำ |
|---|---|---|
| ครึ่งซ้ายของหน้าจอ | แตะ | คะแนนซ้าย +1 |
| ครึ่งขวาของหน้าจอ | แตะ | คะแนนขวา +1 |
| ด้านล่าง ~30% ของหน้าจอ | แตะ | รีเซ็ตตัวจับเวลาผู้รักษาประตู |
| ฝั่งที่ไฮไลต์ (ระหว่างหน้าต่างเลิกทำ 3s) | แตะ | เลิกทำคะแนนล่าสุด |
