# Jalkapallotulostaulukko

Paras kojelauta pelikerhojen ja 5-a-side otteluihisi.

## Ominaisuudet

- **Tulostaulukko:** Napauta vasemmalle (Joukkue A) tai oikealle (Joukkue B) lisataksesi maaleja.
- **Maalivahtitajastin:** Napauta ala-aluetta nollataksesi maalivahtitajastimen (varinalla).
- **Suorituksen tallennus:** Seuraa GPS:aa, syketieto ja tallentaa ottelun jalkapallotoimintona.
- **Vain kosketusnaytto:** Optimoitu Fenix 7/8, Epix, Venu ja Forerunner kosketusmalleille.

## Ohjaus

### Fyysiset painikkeet

| Painike | Toiminto | Mita se tekee |
|---|---|---|
| **START/STOP** (top-right) | Paina | Aloita tai keskeyta tallennus |
| **START/STOP** (top-right) | Paina (taukovalikossa) | Vahvista valittu vaihtoehto (Jatka/Tallenna/Hylkaa) |
| **UP/MENU** (top-left) | Paina | +1 vasen tulos |
| **UP/MENU** (top-left) | Pitka painallus | Siirry tulosten saatotilaan (vasen puoli valittu) |
| **UP/MENU** (top-left) | Paina (tulosten saadossa) | Vaihda vasemman/oikean puolen valilla |
| **DOWN** (bottom-left) | Paina | +1 oikea tulos |
| **DOWN** (bottom-left) | Paina (tulosten saadossa) | −1 valitulta puolelta |
| **DOWN** (bottom-left) | Paina (vahvistusnayotolla) | Takaisin taukovalikkoon |
| **BACK/LAP** (bottom-right) | Paina | Nollaa maalivahtitajastin |
| **BACK/LAP** (bottom-right) | Pitka painallus | Avaa maalivahtitajastimen asetukset |
| **BACK/LAP** (bottom-right) | Paina (tulosten saadossa) | Poistu tulosten saatotilasta |
| **BACK/LAP** (bottom-right) | Paina (vahvistusnayotolla) | Takaisin taukovalikkoon |

### Kosketusnaytonalueet

| Alue | Toiminto | Mita se tekee |
|---|---|---|
| Naytonvasen puolisko | Napauta | +1 vasen tulos |
| Naytonoikea puolisko | Napauta | +1 oikea tulos |
| Naytonalin ~30% | Napauta | Nollaa maalivahtitajastin |
| Korostettu puoli (3s kumoamisikkunan aikana) | Napauta | Kumoa viimeisin tulos |
