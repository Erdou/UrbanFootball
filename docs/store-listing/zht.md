# 足球記分員

適用於街頭足球和5-a-side比賽的終極儀表板。

## 功能

- **記分板:** 點擊左側（A隊）或右側（B隊）新增進球。
- **守門員計時器:** 點擊底部區域重置守門員計時器（附振動）。
- **活動記錄:** 追蹤GPS、心率，並將比賽儲存為足球活動。
- **僅限觸控螢幕:** 針對Fenix 7/8、Epix、Venu和Forerunner觸控型號最佳化。

## 操作方式

### 實體按鈕

| 按鈕 | 操作 | 功能說明 |
|---|---|---|
| **START/STOP** (top-right) | 按下 | 開始或暫停記錄 |
| **START/STOP** (top-right) | 按下（暫停選單中） | 確認所選選項（恢復/儲存/捨棄） |
| **UP/MENU** (top-left) | 按下 | 左側得分 +1 |
| **UP/MENU** (top-left) | 長按 | 進入得分調整模式（選取左側） |
| **UP/MENU** (top-left) | 按下（得分調整中） | 切換左側/右側 |
| **DOWN** (bottom-left) | 按下 | 右側得分 +1 |
| **DOWN** (bottom-left) | 按下（得分調整中） | 所選側得分 −1 |
| **DOWN** (bottom-left) | 按下（確認畫面中） | 返回暫停選單 |
| **BACK/LAP** (bottom-right) | 按下 | 重置守門員計時器 |
| **BACK/LAP** (bottom-right) | 長按 | 開啟守門員計時器設定 |
| **BACK/LAP** (bottom-right) | 按下（得分調整中） | 退出得分調整模式 |
| **BACK/LAP** (bottom-right) | 按下（確認畫面中） | 返回暫停選單 |

### 觸控螢幕區域

| 區域 | 操作 | 功能說明 |
|---|---|---|
| 螢幕左半部分 | 點擊 | 左側得分 +1 |
| 螢幕右半部分 | 點擊 | 右側得分 +1 |
| 螢幕底部 ~30% | 點擊 | 重置守門員計時器 |
| 高亮側（3s 復原視窗期間） | 點擊 | 復原上一次得分 |
