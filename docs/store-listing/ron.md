# Tablou de Scor Fotbal

Panoul suprem pentru meciurile tale amicale și partidele de 5-a-side.

## Caracteristici

- **Tablou de scor:** Atinge stânga (Echipa A) sau dreapta (Echipa B) pentru a adăuga goluri.
- **Cronometru Portar:** Atinge zona inferioară pentru a reseta cronometrul portarului (cu vibrație).
- **Înregistrare Activitate:** Urmărește GPS, frecvența cardiacă și salvează meciul ca activitate de Fotbal.
- **Doar ecran tactil:** Optimizat pentru Fenix 7/8, Epix, Venu și modelele tactile Forerunner.

## Comenzi

### Butoane fizice

| Buton | Acțiune | Funcție |
|---|---|---|
| **START/STOP** (top-right) | Apăsare | Pornește sau pune pe pauză înregistrarea |
| **START/STOP** (top-right) | Apăsare (în meniul de pauză) | Confirmă opțiunea selectată (Reia/Salvează/Renunță) |
| **UP/MENU** (top-left) | Apăsare | +1 scor stânga |
| **UP/MENU** (top-left) | Apăsare lungă | Intră în modul Ajustare Scor (partea stângă selectată) |
| **UP/MENU** (top-left) | Apăsare (în Ajustare Scor) | Comută între partea stângă/dreaptă |
| **DOWN** (bottom-left) | Apăsare | +1 scor dreapta |
| **DOWN** (bottom-left) | Apăsare (în Ajustare Scor) | −1 pe partea selectată |
| **DOWN** (bottom-left) | Apăsare (pe ecranul de confirmare) | Înapoi la meniul de pauză |
| **BACK/LAP** (bottom-right) | Apăsare | Resetează cronometrul portarului |
| **BACK/LAP** (bottom-right) | Apăsare lungă | Deschide configurarea cronometrului portarului |
| **BACK/LAP** (bottom-right) | Apăsare (în Ajustare Scor) | Iese din modul Ajustare Scor |
| **BACK/LAP** (bottom-right) | Apăsare (pe ecranul de confirmare) | Înapoi la meniul de pauză |

### Zone tactile

| Zonă | Acțiune | Funcție |
|---|---|---|
| Jumătatea stângă a ecranului | Atingere | +1 scor stânga |
| Jumătatea dreaptă a ecranului | Atingere | +1 scor dreapta |
| ~30% inferioară a ecranului | Atingere | Resetează cronometrul portarului |
| Partea evidențiată (în fereastra de anulare de 3s) | Atingere | Anulează ultimul scor |
