# Segnapunti Calcio

Il pannello definitivo per le tue partite improvvisate e i match a 5-a-side.

## Funzionalità

- **Tabellone:** Tocca a sinistra (Squadra A) o a destra (Squadra B) per aggiungere gol.
- **Timer Portiere:** Tocca la zona inferiore per reimpostare il timer del portiere (con vibrazione).
- **Registrazione Attività:** Traccia GPS, frequenza cardiaca e salva la partita come attività Calcio.
- **Solo touchscreen:** Ottimizzato per Fenix 7/8, Epix, Venu e modelli touch Forerunner.

## Comandi

### Pulsanti fisici

| Pulsante | Azione | Funzione |
|---|---|---|
| **START/STOP** (top-right) | Premere | Avviare o mettere in pausa la registrazione |
| **START/STOP** (top-right) | Premere (nel menu pausa) | Confermare l'opzione selezionata (Riprendi/Salva/Elimina) |
| **UP/MENU** (top-left) | Premere | +1 punteggio sinistro |
| **UP/MENU** (top-left) | Pressione lunga | Entrare in modalità Regolazione Punteggio (lato sinistro selezionato) |
| **UP/MENU** (top-left) | Premere (in Regolazione Punteggio) | Alternare tra lato sinistro/destro |
| **DOWN** (bottom-left) | Premere | +1 punteggio destro |
| **DOWN** (bottom-left) | Premere (in Regolazione Punteggio) | −1 sul lato selezionato |
| **DOWN** (bottom-left) | Premere (nella schermata di conferma) | Tornare al menu pausa |
| **BACK/LAP** (bottom-right) | Premere | Reimpostare il timer portiere |
| **BACK/LAP** (bottom-right) | Pressione lunga | Aprire la configurazione del timer portiere |
| **BACK/LAP** (bottom-right) | Premere (in Regolazione Punteggio) | Uscire dalla modalità Regolazione Punteggio |
| **BACK/LAP** (bottom-right) | Premere (nella schermata di conferma) | Tornare al menu pausa |

### Zone touchscreen

| Zona | Azione | Funzione |
|---|---|---|
| Metà sinistra dello schermo | Tocco | +1 punteggio sinistro |
| Metà destra dello schermo | Tocco | +1 punteggio destro |
| ~30% inferiore dello schermo | Tocco | Reimpostare il timer portiere |
| Lato evidenziato (durante la finestra di annullamento di 3s) | Tocco | Annullare l'ultimo punteggio |
