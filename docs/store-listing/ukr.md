# Football Scorekeeper

Найкраща панель для ваших дружніх матчів та ігор 5-a-side.

## Функції

- **Табло:** Торкніться лівої сторони (Команда A) або правої (Команда B), щоб додати гол.
- **Таймер воротаря:** Торкніться нижньої частини, щоб скинути таймер воротаря (з вібрацією).
- **Запис активності:** Відстежує GPS, пульс і зберігає матч як футбольну активність.
- **Лише сенсорний екран:** Оптимізовано для Fenix 7/8, Epix, Venu та сенсорних моделей Forerunner.

## Керування

### Фізичні кнопки

| Кнопка | Дія | Що робить |
|---|---|---|
| **START/STOP** (top-right) | Натисніть | Почати або призупинити запис |
| **START/STOP** (top-right) | Натисніть (у меню паузи) | Підтвердити обраний варіант (Продовжити/Зберегти/Скасувати) |
| **UP/MENU** (top-left) | Натисніть | +1 лівий рахунок |
| **UP/MENU** (top-left) | Довге натискання | Увійти в режим коригування рахунку (обрано ліву сторону) |
| **UP/MENU** (top-left) | Натисніть (у коригуванні рахунку) | Перемикання між лівою/правою стороною |
| **DOWN** (bottom-left) | Натисніть | +1 правий рахунок |
| **DOWN** (bottom-left) | Натисніть (у коригуванні рахунку) | −1 на обраній стороні |
| **DOWN** (bottom-left) | Натисніть (на екрані підтвердження) | Назад до меню паузи |
| **BACK/LAP** (bottom-right) | Натисніть | Скинути таймер воротаря |
| **BACK/LAP** (bottom-right) | Довге натискання | Відкрити налаштування таймера воротаря |
| **BACK/LAP** (bottom-right) | Натисніть (у коригуванні рахунку) | Вийти з режиму коригування рахунку |
| **BACK/LAP** (bottom-right) | Натисніть (на екрані підтвердження) | Назад до меню паузи |

### Зони сенсорного екрана

| Зона | Дія | Що робить |
|---|---|---|
| Ліва половина екрана | Торкніться | +1 лівий рахунок |
| Права половина екрана | Торкніться | +1 правий рахунок |
| Нижні ~30% екрана | Торкніться | Скинути таймер воротаря |
| Виділена сторона (під час 3s вікна скасування) | Торкніться | Скасувати останнє очко |
