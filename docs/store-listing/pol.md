# Football Scorekeeper

Najlepszy panel do twoich meczow towarzyskich i gier 5-a-side.

## Funkcje

- **Tablica wynikow:** Dotknij lewej strony (Druzyna A) lub prawej (Druzyna B), aby dodac gola.
- **Timer bramkarza:** Dotknij dolnej czesci ekranu, aby zresetowac timer bramkarza (z wibracja).
- **Nagrywanie aktywnosci:** Sledzi GPS, tetno i zapisuje mecz jako aktywnosc pilkarska.
- **Tylko ekran dotykowy:** Zoptymalizowany dla Fenix 7/8, Epix, Venu i dotykowych modeli Forerunner.

## Sterowanie

### Przyciski fizyczne

| Przycisk | Akcja | Co robi |
|---|---|---|
| **START/STOP** (top-right) | Nacisnij | Rozpocznij lub wstrzymaj nagrywanie |
| **START/STOP** (top-right) | Nacisnij (w menu pauzy) | Potwierdz wybrana opcje (Wznow/Zapisz/Odrzuc) |
| **UP/MENU** (top-left) | Nacisnij | +1 lewy wynik |
| **UP/MENU** (top-left) | Dlugie nacisniecie | Wejdz w tryb korekty wyniku (wybrana lewa strona) |
| **UP/MENU** (top-left) | Nacisnij (w korekcie wyniku) | Przelacz miedzy lewa/prawa strona |
| **DOWN** (bottom-left) | Nacisnij | +1 prawy wynik |
| **DOWN** (bottom-left) | Nacisnij (w korekcie wyniku) | −1 po wybranej stronie |
| **DOWN** (bottom-left) | Nacisnij (na ekranie potwierdzenia) | Wroc do menu pauzy |
| **BACK/LAP** (bottom-right) | Nacisnij | Resetuj timer bramkarza |
| **BACK/LAP** (bottom-right) | Dlugie nacisniecie | Otworz konfiguracje timera bramkarza |
| **BACK/LAP** (bottom-right) | Nacisnij (w korekcie wyniku) | Wyjdz z trybu korekty wyniku |
| **BACK/LAP** (bottom-right) | Nacisnij (na ekranie potwierdzenia) | Wroc do menu pauzy |

### Strefy ekranu dotykowego

| Strefa | Akcja | Co robi |
|---|---|---|
| Lewa polowa ekranu | Dotknij | +1 lewy wynik |
| Prawa polowa ekranu | Dotknij | +1 prawy wynik |
| Dolne ~30% ekranu | Dotknij | Resetuj timer bramkarza |
| Podswietlona strona (podczas 3s okna cofania) | Dotknij | Cofnij ostatni punkt |
