# Placar de Futebol

O painel definitivo para os seus jogos informais e partidas de 5-a-side.

## Funcionalidades

- **Placar:** Toque à esquerda (Equipa A) ou à direita (Equipa B) para adicionar golos.
- **Temporizador do Guarda-Redes:** Toque na zona inferior para reiniciar o temporizador do guarda-redes (com vibração).
- **Gravação de Atividade:** Regista GPS, frequência cardíaca e guarda o jogo como atividade de Futebol.
- **Apenas ecrã tátil:** Otimizado para Fenix 7/8, Epix, Venu e modelos táteis Forerunner.

## Controlos

### Botões físicos

| Botão | Ação | Função |
|---|---|---|
| **START/STOP** (top-right) | Premir | Iniciar ou pausar a gravação |
| **START/STOP** (top-right) | Premir (no menu de pausa) | Confirmar a opção selecionada (Retomar/Guardar/Descartar) |
| **UP/MENU** (top-left) | Premir | +1 pontuação esquerda |
| **UP/MENU** (top-left) | Pressão longa | Entrar no modo Ajuste de Pontuação (lado esquerdo selecionado) |
| **UP/MENU** (top-left) | Premir (em Ajuste de Pontuação) | Alternar entre lado esquerdo/direito |
| **DOWN** (bottom-left) | Premir | +1 pontuação direita |
| **DOWN** (bottom-left) | Premir (em Ajuste de Pontuação) | −1 no lado selecionado |
| **DOWN** (bottom-left) | Premir (no ecrã de confirmação) | Voltar ao menu de pausa |
| **BACK/LAP** (bottom-right) | Premir | Reiniciar temporizador do guarda-redes |
| **BACK/LAP** (bottom-right) | Pressão longa | Abrir configuração do temporizador do guarda-redes |
| **BACK/LAP** (bottom-right) | Premir (em Ajuste de Pontuação) | Sair do modo Ajuste de Pontuação |
| **BACK/LAP** (bottom-right) | Premir (no ecrã de confirmação) | Voltar ao menu de pausa |

### Zonas táteis

| Zona | Ação | Função |
|---|---|---|
| Metade esquerda do ecrã | Toque | +1 pontuação esquerda |
| Metade direita do ecrã | Toque | +1 pontuação direita |
| ~30% inferior do ecrã | Toque | Reiniciar temporizador do guarda-redes |
| Lado destacado (durante a janela de anulação de 3s) | Toque | Anular a última pontuação |
